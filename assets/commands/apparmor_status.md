# TAGLINE

Report the current AppArmor confinement state.

# TLDR

This command is an **alias** of aa-status

View documentation for the original command

```tldr aa-status```

# SYNOPSIS

**apparmor_status**

# DESCRIPTION

**apparmor_status** is an alias for **aa-status**, which reports various aspects of the current state of AppArmor confinement. See **aa-status** for full documentation.

# INSTALL

```apt: sudo apt install apparmor```

```pacman: sudo pacman -S apparmor```

```apk: sudo apk add apparmor-utils```

```zypper: sudo zypper install apparmor-utils```

```nix: nix profile install nixpkgs#apparmor-utils```
