# TAGLINE

Text-based backgammon board game.

# TLDR

**Start** backgammon game

```backgammon```

# SYNOPSIS

**backgammon**

# DESCRIPTION

**backgammon** is a classic board game included in BSD games. It provides a text-based interface for playing backgammon against the computer.

The game implements standard backgammon rules with simple command-line interaction.

# GAMEPLAY

Players move checkers based on dice rolls, attempting to bear off all pieces before their opponent. The computer opponent provides challenging gameplay.

# CAVEATS

Text-based interface only. Limited AI difficulty options. Part of bsd-games package on most systems.

# HISTORY

**backgammon** has been part of BSD games since the early days of Unix, dating back to **1980s** BSD distributions.
