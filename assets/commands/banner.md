# TAGLINE

Print large ASCII art text

# TLDR

**Print** banner text

```banner [HELLO]```

# SYNOPSIS

**banner** _text_

# DESCRIPTION

**banner** prints large ASCII art banner text to the terminal. It converts text into large block letters suitable for creating eye-catching messages or headers in terminal output.

The tool is useful for creating visual separators in scripts or displaying prominent messages.

# FEATURES

Each character is displayed in large format using hash symbols (#) or asterisks. Output is typically several lines tall per character.

# CAVEATS

Limited to uppercase letters and simple characters. Output width can be very large. Not all systems have banner installed. Modern alternatives (figlet, toilet) offer more fonts and features.

# HISTORY

**banner** has been part of Unix systems since the early days, originally used for printing large banners on line printers in the **1970s**.
