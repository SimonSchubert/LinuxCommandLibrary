# TLDR

**Start** Bastet Tetris game

```bastet```

# SYNOPSIS

**bastet**

# DESCRIPTION

**bastet** (Bastard Tetris) is a Tetris clone with a twist - it chooses the worst possible piece for your current situation. The game analyzes your board and deliberately gives you pieces that are hardest to place, making it significantly more challenging than standard Tetris.

The game provides a humorous and difficult take on the classic puzzle game.

# FEATURES

- "Adversarial" piece selection algorithm
- Classic Tetris gameplay
- High score tracking
- Terminal-based interface
- ncurses graphics

# CONTROLS

- **Left/Right arrows** - Move piece
- **Up arrow** - Rotate
- **Down arrow** - Soft drop
- **Space** - Hard drop
- **p** - Pause
- **q** - Quit

# CAVEATS

Significantly harder than normal Tetris. Requires terminal with color support. Part of some game packages. May be frustrating by design.

# HISTORY

**bastet** was created by Federico Poloni around **2005** as a deliberately difficult variant of Tetris using an algorithm to select worst-case pieces.
