# SYNOPSIS

**battlestar**

# DESCRIPTION

**battlestar** is a classic text adventure game included in BSD games. Players explore locations, solve puzzles, and interact with characters in a science fiction setting.

The game is a traditional interactive fiction with parser-based input.

# GAMEPLAY

Players type commands to navigate and interact with the game world. Common commands include directions (north, south, etc.), examine, take, use, and talk.

# COMMANDS

Typical adventure game commands:
- **look** - Examine surroundings
- **inventory** - Check items
- **take/drop** - Manage items
- **north/south/east/west** - Movement
- **save/restore** - Save game state

# CAVEATS

Text-only interface. Requires understanding of adventure game conventions. Parser can be finicky. Part of bsd-games package.

# HISTORY

**battlestar** was created in the early **1980s** during the golden age of text adventures, inspired by games like Adventure and Zork.
