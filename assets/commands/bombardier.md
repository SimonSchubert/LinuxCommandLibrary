# DESCRIPTION

**bombardier** is a terminal-based arcade game inspired by classic bomber games. The player flies a plane across the screen, dropping bombs to destroy buildings below. The plane descends with each pass, and the player must clear a path before crashing into the remaining structures.

The game uses simple controls: press space to drop a bomb. Buildings are generated randomly each round.

# CAVEATS

Requires a terminal with ncurses support. Simple but addictive gameplay.

# HISTORY

**bombardier** is inspired by classic bomber games dating back to the **1970s**, particularly Blitz and Bomber on early home computers.
