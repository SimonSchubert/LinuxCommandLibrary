# TLDR

Display **version**

```btrfs version```

Display **help**

```btrfs version --help```

# SYNOPSIS

**btrfs version**

# DESCRIPTION

**btrfs version** displays the version of btrfs-progs, the userspace utilities for btrfs filesystem management. This is useful for determining which features are available and for troubleshooting.
