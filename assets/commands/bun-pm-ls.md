# TLDR

**List installed packages**

```bun pm ls```

**List all packages** including nested dependencies

```bun pm ls --all```

# SYNOPSIS

**bun** **pm** **ls** [_options_]

# PARAMETERS

**--all**
> Show all dependencies including nested.

# DESCRIPTION

**bun pm ls** lists packages installed in the current project. This is equivalent to **bun list**.
