# TLDR

**Compare two environments**

```conda compare [env_name] [environment.yml]```

**Compare current environment** with a spec file

```conda compare [environment.yml]```

# SYNOPSIS

**conda** **compare** [_env_name_] _file_

# DESCRIPTION

**conda compare** compares packages in an environment against a specification file (environment.yml). It shows which packages differ between the current environment and the specification.

Useful for checking if an environment matches a specification or needs updates.
