# TLDR

**Deactivate the current environment**

```conda deactivate```

# SYNOPSIS

**conda** **deactivate**

# DESCRIPTION

**conda deactivate** deactivates the currently active conda environment, returning to the base environment or the previously active environment if nested.

The shell PATH and environment variables are restored to their previous state.
