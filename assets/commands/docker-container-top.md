# TLDR

**Show running processes** in a container

```docker container top [container]```

**Show with custom ps options**

```docker container top [container] aux```

# SYNOPSIS

**docker** **container** **top** _container_ [_ps options_]

# DESCRIPTION

**docker container top** displays the running processes inside a container. Accepts ps options to customize output format.
