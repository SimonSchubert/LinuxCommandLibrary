# TAGLINE

Stop one or more running Docker containers (alias)

# TLDR

**Stop a container** (alias of `docker container stop`)

```docker stop mycontainer```

**See** `docker container stop` for full options and details

```tldr docker container stop```

# SYNOPSIS

**docker** **stop** [_options_] _container_ [_container_...]

# DESCRIPTION

**docker stop** is an alias for `docker container stop`. It stops running containers by sending a termination signal and waiting for graceful shutdown (or timeout).

# SEE ALSO

docker-container-stop, docker-kill
