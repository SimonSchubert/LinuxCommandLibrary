# TLDR

**List projects**

```doppler projects```

**Create a project**

```doppler projects create [name]```

**Get project info**

```doppler projects get [name]```

**Delete a project**

```doppler projects delete [name]```

**Update project description**

```doppler projects update [name] --description "[description]"```

# SYNOPSIS

**doppler** **projects** [_command_] [_options_]

# SUBCOMMANDS

**list**
> List all projects.

**create**
> Create a project.

**get**
> Get project details.

**update**
> Update project.

**delete**
> Delete a project.

# DESCRIPTION

**doppler projects** manages Doppler projects. Projects contain environments and their associated secrets and configurations.
