# TLDR

**Convert ESCP2 to PBM**

```escp2topbm < [input.escp2] > [output.pbm]```

# SYNOPSIS

**escp2topbm** [_options_] [_input_]

# DESCRIPTION

**escp2topbm** converts Epson ESC/P2 printer output to PBM (Portable Bitmap) format. Part of the Netpbm image processing toolkit.
