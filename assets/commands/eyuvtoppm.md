# TLDR

**Convert EYUV to PPM**

```eyuvtoppm -w [width] -h [height] < [input.eyuv] > [output.ppm]```

# SYNOPSIS

**eyuvtoppm** [_options_] [_input_]

# PARAMETERS

**-w** _width_
> Image width.

**-h** _height_
> Image height.

# DESCRIPTION

**eyuvtoppm** converts Encoder/Berkeley YUV video format to PPM (Portable Pixmap) format. Part of the Netpbm image processing toolkit.
