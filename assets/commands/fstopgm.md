# TLDR

**Convert Usenix FaceSaver to PGM**

```fstopgm < [input.fs] > [output.pgm]```

# SYNOPSIS

**fstopgm** [_input_]

# DESCRIPTION

**fstopgm** converts Usenix FaceSaver format images to PGM (Portable Graymap) format. Part of the Netpbm image processing toolkit.
