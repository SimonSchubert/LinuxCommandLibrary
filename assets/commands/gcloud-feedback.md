# TLDR

**Send feedback to Google**

```gcloud feedback```

**Include log files**

```gcloud feedback --log-file [log_path]```

# SYNOPSIS

**gcloud** **feedback** [_options_]

# PARAMETERS

**--log-file** _file_
> Attach log file to feedback.

# DESCRIPTION

**gcloud feedback** opens a browser to send feedback to Google Cloud. Provides a way to report issues or suggest improvements for the gcloud CLI.
