# TLDR

**Initialize gcloud**

```gcloud init```

**Initialize without browser**

```gcloud init --console-only```

**Reinitialize with new configuration**

```gcloud init --skip-diagnostics```

# SYNOPSIS

**gcloud** **init** [_options_]

# PARAMETERS

**--console-only**
> Don't launch browser for login.

**--skip-diagnostics**
> Skip network diagnostics.

**--configuration** _name_
> Use specific configuration.

# DESCRIPTION

**gcloud init** initializes the Google Cloud SDK. Guides through authentication, project selection, and default region/zone configuration.
