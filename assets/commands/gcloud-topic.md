# TAGLINE

displays supplementary help topics

# TLDR

**List available topics**

```gcloud topic```

**View specific topic**

```gcloud topic [configurations]```

**View auth topic**

```gcloud topic [auth]```

**View formats topic**

```gcloud topic [formats]```

# SYNOPSIS

**gcloud** **topic** [_topic_]

# TOPICS

**configurations**
> Named configurations.

**auth**
> Authentication information.

**formats**
> Output formats.

**filters**
> Filter expressions.

**projections**
> Resource projections.

**escaping**
> Escaping and quoting.

**startup**
> SDK startup.

# DESCRIPTION

**gcloud topic** displays supplementary help topics for gcloud commands. Topics provide detailed information about various gcloud concepts and features.
