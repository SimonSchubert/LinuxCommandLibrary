# TLDR

**Show help**

```gcrane help```

**Help for specific command**

```gcrane help [command]```

# SYNOPSIS

**gcrane** **help** [_command_]

# DESCRIPTION

**gcrane help** displays help information for gcrane commands. Shows usage, available subcommands, and options.
