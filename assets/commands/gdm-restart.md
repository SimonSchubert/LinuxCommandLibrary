# TLDR

**Restart GNOME Display Manager**

```sudo gdm-restart```

# SYNOPSIS

**gdm-restart**

# DESCRIPTION

**gdm-restart** restarts the GNOME Display Manager. This will log out all graphical sessions and restart the login screen.

# CAVEATS

Restarting GDM will terminate all user sessions. Save your work before running this command.
