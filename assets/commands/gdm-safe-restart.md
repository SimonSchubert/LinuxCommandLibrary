# TLDR

**Safely restart GDM**

```sudo gdm-safe-restart```

# SYNOPSIS

**gdm-safe-restart**

# DESCRIPTION

**gdm-safe-restart** safely restarts the GNOME Display Manager when no users are logged in. Will not restart if users have active sessions.
