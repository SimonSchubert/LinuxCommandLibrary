# TLDR

**Stop GNOME Display Manager**

```sudo gdm-stop```

# SYNOPSIS

**gdm-stop**

# DESCRIPTION

**gdm-stop** stops the GNOME Display Manager service. This terminates the graphical login interface and returns to a text console.

# CAVEATS

Stopping GDM will terminate all graphical sessions. Save your work before running this command.
