# TLDR

**Convert GEM IMG to PBM**

```gemtopbm [input.img] > [output.pbm]```

# SYNOPSIS

**gemtopbm** [_input_]

# DESCRIPTION

**gemtopbm** converts GEM IMG format images to PBM (Portable Bitmap) format. GEM IMG is an older bitmap format from Digital Research GEM systems.
