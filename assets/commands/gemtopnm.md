# TAGLINE

GEM IMG to PNM converter

# TLDR

**Convert GEM IMG to PNM**

```gemtopnm [input.img] > [output.pnm]```

# SYNOPSIS

**gemtopnm** [_input_]

# DESCRIPTION

**gemtopnm** converts GEM IMG format images to PNM (Portable Any Map) format. Handles both monochrome and color GEM images.
