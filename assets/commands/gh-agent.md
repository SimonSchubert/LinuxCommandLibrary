# TLDR

**List available agents**

```gh agent list```

**Run an agent**

```gh agent run [agent_name]```

**Get agent info**

```gh agent info [agent_name]```

# SYNOPSIS

**gh** **agent** _command_ [_options_]

# SUBCOMMANDS

**list**
> List available agents.

**run**
> Execute an agent.

**info**
> Show agent details.

# DESCRIPTION

**gh agent** manages GitHub Copilot agents (extensions). Agents extend Copilot's capabilities with specialized knowledge and tools.
