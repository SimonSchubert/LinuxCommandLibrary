# TLDR

**List environments**

```gh environment list```

**View environment details**

```gh environment view [env_name]```

# SYNOPSIS

**gh** **environment** _command_ [_options_]

# SUBCOMMANDS

**list**
> List environments.

**view**
> View environment details.

# PARAMETERS

**-R**, **--repo** _owner/repo_
> Repository.

# DESCRIPTION

**gh environment** manages GitHub repository environments. Environments are used for deployment targets with protection rules and secrets.
