# TLDR

**Preview markdown**

```gh preview [file.md]```

**Preview from stdin**

```echo "[# Heading]" | gh preview```

# SYNOPSIS

**gh** **preview** [_file_]

# DESCRIPTION

**gh preview** renders Markdown files in the terminal. Useful for previewing READMEs and documentation before committing.
