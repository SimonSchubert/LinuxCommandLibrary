# TLDR

**List rulesets**

```gh ruleset list```

**View a ruleset**

```gh ruleset view [ruleset_id]```

**Check branch rules**

```gh ruleset check [branch]```

# SYNOPSIS

**gh** **ruleset** _command_ [_options_]

# SUBCOMMANDS

**list**
> List rulesets.

**view**
> View a ruleset.

**check**
> Check rules for branch.

# PARAMETERS

**-R**, **--repo** _owner/repo_
> Repository.

**--org** _name_
> Organization.

# DESCRIPTION

**gh ruleset** manages GitHub repository rulesets. Rulesets define branch protection rules, status checks, and other repository policies.
