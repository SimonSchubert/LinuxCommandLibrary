# TLDR

**Start screensaver**

```gh screensaver```

**Start specific screensaver**

```gh screensaver -s [fireworks]```

# SYNOPSIS

**gh** **screensaver** [_options_]

# PARAMETERS

**-s**, **--saver** _name_
> Screensaver name.

# DESCRIPTION

**gh screensaver** displays animated terminal screensavers. A fun Easter egg feature in the GitHub CLI.
