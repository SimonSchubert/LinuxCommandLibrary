# TLDR

**View contribution skyline**

```gh skyline```

**View for specific user**

```gh skyline [username]```

**View specific year**

```gh skyline --year [2023]```

# SYNOPSIS

**gh** **skyline** [_username_] [_options_]

# PARAMETERS

**--year** _year_
> Year to display.

# DESCRIPTION

**gh skyline** displays a 3D visualization of GitHub contributions, similar to the GitHub Skyline feature. Shows commit activity as buildings on a skyline.
