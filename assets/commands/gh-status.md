# TLDR

**Show status dashboard**

```gh status```

**Show in specific organization**

```gh status -o [org_name]```

**Exclude repository**

```gh status -e [owner/repo]```

# SYNOPSIS

**gh** **status** [_options_]

# PARAMETERS

**-o**, **--org** _name_
> Limit to organization.

**-e**, **--exclude** _repo_
> Exclude repository.

# DESCRIPTION

**gh status** shows a dashboard of your GitHub activity including assigned issues, review requests, mentions, and repository activity.
