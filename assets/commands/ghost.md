# TLDR

**Start Ghost in development**

```ghost start```

**Start in production**

```ghost start --production```

**Stop Ghost**

```ghost stop```

**Restart Ghost**

```ghost restart```

**Check status**

```ghost status```

**Install Ghost**

```ghost install```

**Update Ghost**

```ghost update```

# SYNOPSIS

**ghost** _command_ [_options_]

# SUBCOMMANDS

**install**
> Install Ghost.

**start**
> Start Ghost.

**stop**
> Stop Ghost.

**restart**
> Restart Ghost.

**status**
> Show status.

**update**
> Update Ghost.

**setup**
> Configure Ghost.

**config**
> View/set configuration.

**log**
> View logs.

**doctor**
> Check installation.

# PARAMETERS

**--production**
> Run in production mode.

**-d**, **--dir** _path_
> Installation directory.

# DESCRIPTION

**ghost** is the CLI for Ghost, an open-source publishing platform. Manages Ghost installations, updates, and configurations.
