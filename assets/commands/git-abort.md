# TLDR

**Abort current operation**

```git abort```

# SYNOPSIS

**git** **abort**

# DESCRIPTION

**git abort** aborts the current in-progress operation like rebase, merge, or cherry-pick. Part of git-extras, it detects and aborts any ongoing operation automatically.
