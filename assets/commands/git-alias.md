# TLDR

**List all aliases**

```git alias```

**Search aliases**

```git alias [pattern]```

**Create alias**

```git alias [name] "[command]"```

# SYNOPSIS

**git** **alias** [_pattern_]

**git** **alias** _name_ _command_

# DESCRIPTION

**git alias** lists and creates Git aliases. Part of git-extras. When called without arguments, lists all defined aliases. With one argument, searches aliases. With two arguments, creates an alias.
