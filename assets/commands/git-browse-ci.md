# TLDR

**Open CI page in browser**

```git browse-ci```

# SYNOPSIS

**git** **browse-ci**

# DESCRIPTION

**git browse-ci** opens the CI/CD pipeline page in a web browser. Part of git-extras. Works with GitHub Actions, GitLab CI, and other CI systems.
