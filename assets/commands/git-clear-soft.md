# TLDR

**Soft clear working directory**

```git clear-soft```

# SYNOPSIS

**git** **clear-soft**

# DESCRIPTION

**git clear-soft** performs a soft reset, keeping changes staged. Part of git-extras. Useful for reorganizing commits without losing work.
