# TLDR

**Clear working directory**

```git clear```

**Clear with force**

```git clear -f```

# SYNOPSIS

**git** **clear** [_options_]

# PARAMETERS

**-f**, **--force**
> Force without confirmation.

# DESCRIPTION

**git clear** removes untracked files and resets the working directory. Part of git-extras. Combines git reset and git clean operations.
