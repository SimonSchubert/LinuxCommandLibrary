# TLDR

**Launch Git Cola**

```git cola```

**Open with specific repository**

```git cola [path/to/repo]```

**Open in amend mode**

```git cola --amend```

**Open specific dialog**

```git cola stash```

# SYNOPSIS

**git** **cola** [_options_] [_path_]

# PARAMETERS

**--amend**
> Start in amend mode.

**--version**
> Show version.

**--prompt**
> Prompt for repository.

# DESCRIPTION

**git cola** is a graphical Git client. Provides a visual interface for staging, committing, diffing, and other Git operations. Written in Python using Qt.
