# TLDR

**Copy file with history**

```git cp [source] [dest]```

# SYNOPSIS

**git** **cp** _source_ _dest_

# DESCRIPTION

**git cp** copies a file while preserving Git history. Part of git-extras. Useful when you want to create a new file based on an existing one while keeping the file's history traceable.
