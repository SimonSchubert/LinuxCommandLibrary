# TLDR

**Delete local and remote branch**

```git delete-branch [branch]```

# SYNOPSIS

**git** **delete-branch** _branch_

# DESCRIPTION

**git delete-branch** deletes a branch both locally and on the remote. Part of git-extras. Simplifies branch cleanup by handling both in one command.
