# TLDR

**Delete merged branches**

```git delete-merged-branches```

# SYNOPSIS

**git** **delete-merged-branches**

# DESCRIPTION

**git delete-merged-branches** deletes all branches that have been merged into the current branch. Part of git-extras. Useful for cleaning up after merge workflows.
