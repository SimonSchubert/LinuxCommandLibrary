# TLDR

**Delete squash-merged branches**

```git delete-squashed-branches```

# SYNOPSIS

**git** **delete-squashed-branches**

# DESCRIPTION

**git delete-squashed-branches** deletes branches that have been squash-merged into the current branch. Part of git-extras. Detects squashed branches that won't show as merged with --merged flag.
