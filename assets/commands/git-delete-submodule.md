# TLDR

**Delete a submodule**

```git delete-submodule [path/to/submodule]```

# SYNOPSIS

**git** **delete-submodule** _path_

# DESCRIPTION

**git delete-submodule** completely removes a submodule. Part of git-extras. Handles all cleanup steps: deinit, remove from .gitmodules, remove from .git/config, and delete files.
