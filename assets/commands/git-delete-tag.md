# TLDR

**Delete local and remote tag**

```git delete-tag [tag_name]```

# SYNOPSIS

**git** **delete-tag** _tag_

# DESCRIPTION

**git delete-tag** deletes a tag both locally and on the remote. Part of git-extras. Simplifies tag removal by handling both operations.
