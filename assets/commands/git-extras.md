# TLDR

**Install git-extras**

```git extras --version```

**Update git-extras**

```git extras update```

**List available commands**

```git extras --help```

# SYNOPSIS

**git** **extras** [_options_]

# PARAMETERS

**--version**
> Show version.

**--help**
> Show available commands.

**update**
> Update git-extras.

# DESCRIPTION

**git extras** is a collection of useful Git utilities. Provides additional commands like git-summary, git-changelog, git-delete-branch, and many more to enhance Git workflows.
