# TLDR

**Create feature branch**

```git feature [feature_name]```

**Finish feature branch**

```git feature finish [feature_name]```

# SYNOPSIS

**git** **feature** [_options_] _name_

# SUBCOMMANDS

**finish**
> Merge and delete feature branch.

# DESCRIPTION

**git feature** creates and manages feature branches. Part of git-extras. Creates branches with "feature/" prefix and provides helpers for the feature branch workflow.
