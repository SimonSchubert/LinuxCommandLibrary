# TLDR

**Create orphan branch**

```git fresh-branch [branch_name]```

# SYNOPSIS

**git** **fresh-branch** _name_

# DESCRIPTION

**git fresh-branch** creates a new branch with no history (orphan branch). Part of git-extras. Useful for starting documentation branches or gh-pages.
