# TLDR

**Create gh-pages branch**

```git gh-pages```

# SYNOPSIS

**git** **gh-pages**

# DESCRIPTION

**git gh-pages** creates and sets up a gh-pages branch for GitHub Pages. Part of git-extras. Creates an orphan branch suitable for hosting static websites.
