# TLDR

**Show local commits not pushed**

```git local-commits```

# SYNOPSIS

**git** **local-commits**

# DESCRIPTION

**git local-commits** lists commits on the current branch that haven't been pushed to the remote. Part of git-extras. Useful for reviewing work before pushing.
