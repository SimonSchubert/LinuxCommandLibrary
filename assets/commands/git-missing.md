# TLDR

**Show commits missing from branch**

```git missing [branch]```

**Compare two branches**

```git missing [branch1] [branch2]```

# SYNOPSIS

**git** **missing** [_branch1_] [_branch2_]

# DESCRIPTION

**git missing** shows commits that are in one branch but not the other. Part of git-extras. Useful for comparing branches before merging.
