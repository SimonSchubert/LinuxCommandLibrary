# TLDR

**Create gist from file**

```git paste [file]```

# SYNOPSIS

**git** **paste** _file_

# DESCRIPTION

**git paste** creates a GitHub Gist from a file. Part of git-extras. Uploads the file content and returns the Gist URL.
