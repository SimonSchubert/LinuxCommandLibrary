# TLDR

**Rebase with conflict resolution**

```git psykorebase [branch]```

**Rebase in merge mode**

```git psykorebase --merge [branch]```

# SYNOPSIS

**git** **psykorebase** [_options_] _branch_

# PARAMETERS

**--merge**
> Use merge mode.

# DESCRIPTION

**git psykorebase** performs a rebase that stops to let you fix conflicts, then continues. Part of git-extras. Provides more control during complex rebases.
