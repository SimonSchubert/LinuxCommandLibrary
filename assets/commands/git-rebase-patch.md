# TLDR

**Rebase and apply patch**

```git rebase-patch [patch_file]```

# SYNOPSIS

**git** **rebase-patch** _patch_file_

# DESCRIPTION

**git rebase-patch** rebases a branch while applying a patch. Part of git-extras. Useful for maintaining patches on top of upstream changes.
