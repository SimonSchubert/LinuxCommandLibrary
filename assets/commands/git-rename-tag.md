# TLDR

**Rename a tag**

```git rename-tag [old_name] [new_name]```

# SYNOPSIS

**git** **rename-tag** _old_name_ _new_name_

# DESCRIPTION

**git rename-tag** renames a tag locally and on the remote. Part of git-extras. Creates new tag, deletes old tag, and updates remote.
