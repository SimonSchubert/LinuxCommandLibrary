# TLDR

**Reset file to HEAD**

```git reset-file [file]```

**Reset file to specific commit**

```git reset-file [commit] [file]```

# SYNOPSIS

**git** **reset-file** [_commit_] _file_

# DESCRIPTION

**git reset-file** resets a file to its state at a specific commit. Part of git-extras. Convenient shortcut for resetting individual files.
