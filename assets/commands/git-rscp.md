# TLDR

**Copy files via rsync**

```git rscp [remote] [files]```

# SYNOPSIS

**git** **rscp** _remote_ _files_...

# DESCRIPTION

**git rscp** copies files to a remote using rsync over SSH. Part of git-extras. Uses the remote's SSH configuration from Git.
