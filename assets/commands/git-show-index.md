# TLDR

**Show pack index contents**

```git show-index < [.idx file]```

# SYNOPSIS

**git** **show-index** < _index_file_

# DESCRIPTION

**git show-index** reads a pack index file from stdin and dumps its contents. Shows object offset and SHA-1 for each object in the pack.
