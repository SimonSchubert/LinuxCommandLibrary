# TLDR

**Show merged branches**

```git show-merged-branches```

# SYNOPSIS

**git** **show-merged-branches**

# DESCRIPTION

**git show-merged-branches** lists all branches that have been merged into the current branch. Part of git-extras. Useful for identifying cleanup candidates.
