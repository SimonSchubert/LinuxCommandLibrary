# TAGLINE

List branches not yet merged into the current branch

# TLDR

**Show unmerged branches**

```git show-unmerged-branches```

# SYNOPSIS

**git** **show-unmerged-branches**

# DESCRIPTION

**git show-unmerged-branches** lists all branches that have not been merged into the current branch. Part of git-extras, it helps identify work-in-progress branches and outstanding feature work.
