# TLDR

**Show unmerged branches**

```git show-unmerged-branches```

# SYNOPSIS

**git** **show-unmerged-branches**

# DESCRIPTION

**git show-unmerged-branches** lists all branches that have not been merged into the current branch. Part of git-extras. Helps identify work in progress.
