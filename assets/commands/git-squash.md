# TLDR

**Squash branch into current**

```git squash [branch]```

**Squash with message**

```git squash [branch] "[message]"```

# SYNOPSIS

**git** **squash** _branch_ [_message_]

# DESCRIPTION

**git squash** squashes all commits from a branch into a single commit on the current branch. Part of git-extras. Useful for cleaning up feature branch history before merging.
