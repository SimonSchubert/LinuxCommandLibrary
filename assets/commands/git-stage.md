# TLDR

**Stage files**

```git stage [file]```

**Stage all changes**

```git stage -A```

**Interactive staging**

```git stage -p```

# SYNOPSIS

**git** **stage** [_options_] [_files_...]

# DESCRIPTION

**git stage** is an alias for git add. Stages changes for the next commit. The term "stage" more clearly describes adding changes to the index.
