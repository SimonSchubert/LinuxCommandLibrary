# TLDR

**Unlock a file**

```git unlock [file]```

# SYNOPSIS

**git** **unlock** _file_

# DESCRIPTION

**git unlock** removes the assume-unchanged flag from a file, allowing Git to track changes again. Part of git-extras. Reverses git-lock operation.
