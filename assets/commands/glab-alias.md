# TLDR

**List aliases**

```glab alias list```

**Set alias**

```glab alias set [name] "[command]"```

**Delete alias**

```glab alias delete [name]```

# SYNOPSIS

**glab** **alias** _command_ [_options_]

# SUBCOMMANDS

**list**
> List aliases.

**set**
> Create alias.

**delete**
> Remove alias.

# DESCRIPTION

**glab alias** manages command aliases for the GitLab CLI. Create shortcuts for frequently used commands.
