# TAGLINE

GNOME image viewer and browser

# TLDR

**Open image browser**

```gthumb```

**Open specific folder**

```gthumb [/path/to/photos]```

**Open specific image**

```gthumb [image.jpg]```

**Start slideshow**

```gthumb --slideshow [folder]```

**Import from camera**

```gthumb --import-photos```

# SYNOPSIS

**gthumb** [_options_] [_files_]

# PARAMETERS

_FILES_
> Files or folders to open.

**--slideshow**
> Start slideshow.

**--import-photos**
> Import from device.

**--fullscreen**
> Start in fullscreen.

**-n**, **--new-window**
> Open new window.

**--help**
> Display help information.

# DESCRIPTION

**gThumb** is a GNOME image viewer and browser. It supports viewing, organizing, and editing photos with features like slideshows and batch operations.

The application handles common image formats and provides tools for resizing, cropping, and color adjustment. It integrates with GNOME desktop.

# CAVEATS

GNOME dependencies. Some features require plugins. Memory usage scales with library size.

# HISTORY

gThumb was created by **Paolo Bacchilega** as an image viewer for the **GNOME** desktop environment.
