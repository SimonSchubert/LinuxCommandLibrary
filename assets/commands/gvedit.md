# TLDR

**Open gvedit**

```gvedit```

**Open a graph file**

```gvedit [graph.gv]```

**Open multiple files**

```gvedit [file1.gv] [file2.gv]```

# SYNOPSIS

**gvedit** [_file_ ...]

# DESCRIPTION

**gvedit** is a Qt-based graph editor and viewer for Graphviz. It provides a GUI for creating, viewing, editing, and processing DOT files. Multiple graphs can be open simultaneously, each displayed in its own text window. Features include instant previewing, attribute editing via dialogs, and saving graph settings for future use.
