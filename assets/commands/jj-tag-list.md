# TLDR

**List all tags**

```jj tag list```

**List tags matching pattern**

```jj tag list [pattern]```

# SYNOPSIS

**jj** **tag** **list** [_pattern_]

# DESCRIPTION

**jj tag list** displays all tags in the repository. Optionally filter by a glob pattern. Shows the tag name and the commit it references.
