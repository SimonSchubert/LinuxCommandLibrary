# TLDR

Print **tag information**

```koji taginfo [tag]```

Print info for **multiple tags**

```koji taginfo [tag1] [tag2]```

Display **help**

```koji taginfo --help```

# SYNOPSIS

**koji taginfo** [_options_] _tag_ [_tag_...]

# DESCRIPTION

**koji taginfo** prints basic information about one or more tags including the tag ID, permissions, inheritance, and associated build targets.

# PARAMETERS

**tag**
> Tag name to query (can specify multiple)

**-h, --help**
> Display help information
