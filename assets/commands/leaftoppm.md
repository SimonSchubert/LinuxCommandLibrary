# TLDR

**Convert Interleaf to PPM**

```leaftoppm [input.lf] > [output.ppm]```

# SYNOPSIS

**leaftoppm** [_file_]

# DESCRIPTION

**leaftoppm** converts Interleaf image format to PPM. Interleaf was a document publishing system popular in the 1980s-90s. Part of Netpbm toolkit.
