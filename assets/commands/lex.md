# TLDR

This command is an **alias** of flex

```tldr flex```

# SYNOPSIS

**lex** [_options_] [_file_]

# DESCRIPTION

**lex** is a lexical analyzer generator that is traditionally aliased to **flex** (fast lexical analyzer generator) on most modern systems. It generates C code for tokenizing input based on regular expression patterns.

# INSTALL

```apt: sudo apt install flex```

```dnf: sudo dnf install flex```

```pacman: sudo pacman -S flex```

```apk: sudo apk add flex```

```zypper: sudo zypper install flex```

```brew: brew install flex```

```nix: nix profile install nixpkgs#flex```
