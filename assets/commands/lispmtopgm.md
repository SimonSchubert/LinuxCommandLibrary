# TAGLINE

converts Lisp Machine bitmap format to PGM grayscale

# TLDR

**Convert Lisp Machine bitmap to PGM**

```lispmtopgm [input.lispm] > [output.pgm]```

# SYNOPSIS

**lispmtopgm** [_file_]

# DESCRIPTION

**lispmtopgm** converts Lisp Machine bitmap format to PGM grayscale. Lisp Machines were specialized computers for running Lisp in the 1980s. Part of Netpbm toolkit.
