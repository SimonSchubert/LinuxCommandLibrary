# TAGLINE

decompresses files compressed with lrzip

# TLDR

This command is an **alias** of lrzip --decompress

```tldr lrzip```

# SYNOPSIS

**lrunzip** [_options_] [_file_]

# DESCRIPTION

**lrunzip** decompresses files compressed with lrzip. It is equivalent to running `lrzip --decompress`.

# INSTALL

```apt: sudo apt install lrzip```

```pacman: sudo pacman -S lrzip```

```apk: sudo apk add lrzip```

```zypper: sudo zypper install lrzip```

```brew: brew install lrzip```

```nix: nix profile install nixpkgs#lrzip```
