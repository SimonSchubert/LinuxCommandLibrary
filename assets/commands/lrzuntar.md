# TLDR

This command is an **alias** of lrztar --decompress

```tldr lrztar```

# SYNOPSIS

**lrzuntar** [_options_] _file_

# DESCRIPTION

**lrzuntar** extracts tar archives compressed with lrztar. It is equivalent to running `lrztar --decompress`.
