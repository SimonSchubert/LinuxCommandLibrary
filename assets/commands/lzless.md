# TLDR

**View lzip-compressed file**

```lzless [file.lz]```

# SYNOPSIS

**lzless** [_options_] [_file_...]

# DESCRIPTION

**lzless** displays contents of lzip-compressed files using less. Automatically decompresses for viewing. All less navigation commands work. Press q to quit.
