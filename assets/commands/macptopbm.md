# TLDR

**Convert MacPaint to PBM**

```macptopbm [input.macp] > [output.pbm]```

# SYNOPSIS

**macptopbm** [_file_]

# DESCRIPTION

**macptopbm** converts MacPaint image format to PBM. MacPaint was a bitmap graphics editor for early Macintosh computers. Part of Netpbm toolkit.
