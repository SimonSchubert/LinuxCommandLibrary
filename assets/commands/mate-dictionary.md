# TLDR

**Open dictionary**

```mate-dictionary```

**Look up word**

```mate-dictionary --look-up [word]```

**Look up with specific source**

```mate-dictionary --database [wn] --look-up [word]```

# SYNOPSIS

**mate-dictionary** [_options_]

# PARAMETERS

**--look-up** _word_
> Look up word.

**--database** _db_
> Dictionary database.

**--match** _word_
> Find matching words.

**--strategy** _strategy_
> Matching strategy.

# DESCRIPTION

**mate-dictionary** is a graphical dictionary application for the MATE desktop. Looks up words using DICT protocol servers. Supports multiple dictionaries and search strategies.
