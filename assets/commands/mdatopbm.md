# TAGLINE

converts Microdesign Area format to PBM

# TLDR

**Convert Microdesign Area to PBM**

```mdatopbm [input.mda] > [output.pbm]```

# SYNOPSIS

**mdatopbm** [_options_] [_file_]

# PARAMETERS

**-d**
> Double resolution output.

**-i**
> Invert image colors.

# DESCRIPTION

**mdatopbm** converts Microdesign Area format to PBM. Microdesign was a DTP program for Amstrad PCW computers. Part of Netpbm toolkit.
