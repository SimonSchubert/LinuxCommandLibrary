# TAGLINE

converts MTV ray tracer format to PPM

# TLDR

**Convert MTV ray tracer output to PPM**

```mtvtoppm [input.mtv] > [output.ppm]```

# SYNOPSIS

**mtvtoppm** [_file_]

# DESCRIPTION

**mtvtoppm** converts MTV ray tracer format to PPM. MTV was an early ray tracing program that produced simple image output. Part of Netpbm toolkit.
