# TLDR

**Run window demo**

```muffin-window-demo```

**Demo specific window type**

```muffin-window-demo --type=[dialog]```

# SYNOPSIS

**muffin-window-demo** [_options_]

# PARAMETERS

**--type** _TYPE_
> Window type to demonstrate.

**--help**
> Display help information.

# DESCRIPTION

**muffin-window-demo** demonstrates Muffin window types. It shows different window decorations.

The tool displays sample windows. Used for testing and theme development.

muffin-window-demo shows window types.

# CAVEATS

Development tool. Cinnamon/Muffin specific. For testing window decorations.

# HISTORY

muffin-window-demo is part of **Muffin** development tools for testing window manager behavior.
