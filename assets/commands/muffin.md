# TAGLINE

cinnamon desktop window manager

# TLDR

**Start Muffin window manager**

```muffin```

**Replace running window manager**

```muffin --replace```

**Run in debug mode**

```muffin --debug```

**Display version**

```muffin --version```

# SYNOPSIS

**muffin** [_options_]

# PARAMETERS

**--replace**
> Replace running window manager.

**--debug**
> Enable debug output.

**--version**
> Show version information.

**--help**
> Display help information.

# DESCRIPTION

**muffin** is the Cinnamon desktop window manager. It's a fork of Mutter.

The tool handles window management and compositing. Core component of Cinnamon desktop.

# CAVEATS

Cinnamon desktop specific. Requires compositing support. GPU acceleration recommended.

# HISTORY

muffin was forked from **GNOME's Mutter** for use with the Linux Mint Cinnamon desktop environment.
