# TAGLINE

converts Atari Neochrome format to PPM

# TLDR

**Convert Atari Neochrome to PPM**

```neotoppm [input.neo] > [output.ppm]```

# SYNOPSIS

**neotoppm** [_file_]

# DESCRIPTION

**neotoppm** converts Atari Neochrome format to PPM. Neochrome was a paint program for Atari ST computers. Part of Netpbm toolkit.
