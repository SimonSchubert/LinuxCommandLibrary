# TAGLINE

manages Angular CLI build cache

# TLDR

**Enable cache**

```ng cache enable```

**Disable cache**

```ng cache disable```

**Clear cache**

```ng cache clean```

**Show cache info**

```ng cache info```

# SYNOPSIS

**ng cache** _command_ [_options_]

# PARAMETERS

**enable**
> Enable build cache.

**disable**
> Disable build cache.

**clean**
> Clear cache contents.

**info**
> Display cache information.

# DESCRIPTION

**ng cache** manages Angular CLI build cache. The cache speeds up rebuilds by storing intermediate compilation results. Part of Angular CLI optimization features.
