# TAGLINE

angular CLI

# TLDR

**Create new project**

```ng new [project-name]```

**Serve application**

```ng serve```

**Build application**

```ng build```

**Generate component**

```ng generate component [name]```

**Generate service**

```ng generate service [name]```

**Run tests**

```ng test```

**Run e2e tests**

```ng e2e```

**Add library**

```ng add [package-name]```

# SYNOPSIS

**ng** [_command_] [_options_]

# PARAMETERS

_COMMAND_
> Angular CLI command.

**new** _NAME_
> Create new project.

**serve**
> Start dev server.

**build**
> Build project.

**generate** _TYPE_ _NAME_
> Generate code.

**test**
> Run unit tests.

**--help**
> Display help information.

# DESCRIPTION

**ng** is the Angular CLI. It creates and manages Angular applications.

The tool scaffolds projects. Handles building, testing, and deployment.

# CAVEATS

Node.js required. TypeScript-based. Configuration in angular.json.

# HISTORY

Angular CLI was created by **Google** to provide tooling for Angular framework development.
