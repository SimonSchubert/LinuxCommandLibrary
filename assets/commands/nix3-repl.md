# TLDR

**Start REPL**

```nix3 repl```

**Load nixpkgs**

```nix3 repl '<nixpkgs>'```

**Load flake**

```nix3 repl [.#]```

# SYNOPSIS

**nix3** **repl** [_options_] [_files_]

# PARAMETERS

_FILES_
> Files to load.

**--help**
> Display help information.

# DESCRIPTION

**nix3 repl** starts interactive Nix shell. Alternative name for nix repl.

The tool evaluates expressions. Tab completion and history available.

nix3 repl is interactive shell.

# CAVEATS

Alias for nix repl. :? for help. Expression evaluation.

# HISTORY

nix3 repl provides **interactive evaluation** for Nix expressions and exploration.
