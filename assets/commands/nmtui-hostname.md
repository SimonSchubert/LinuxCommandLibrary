# TAGLINE

Set the system hostname via NetworkManager TUI

# TLDR

This command is an alias of [nmtui](/man/nmtui).

**Launch hostname editor directly**

```nmtui-hostname```

# INSTALL

```apt: sudo apt install network-manager```

```apk: sudo apk add networkmanager-tui```
