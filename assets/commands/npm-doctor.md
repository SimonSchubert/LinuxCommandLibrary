# TAGLINE

runs diagnostics to check npm environment health

# TLDR

**Run npm environment diagnostics**

```npm doctor```

# SYNOPSIS

**npm** **doctor**

# DESCRIPTION

**npm doctor** runs diagnostics to check npm environment health. Verifies npm and node versions, registry connectivity, cache integrity, and permissions. Reports issues and suggests fixes.
