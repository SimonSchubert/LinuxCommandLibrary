# TLDR

**Clean install and test**

```npm install-ci-test```

**Short form**

```npm cit```

# SYNOPSIS

**npm** **install-ci-test**

# DESCRIPTION

**npm install-ci-test** runs npm ci followed by npm test. Combines clean install with test execution. Shorthand for CI pipelines. Alias: npm cit.
