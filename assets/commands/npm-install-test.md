# TAGLINE

runs npm install followed by npm test

# TLDR

**Install and test**

```npm install-test```

**Short form**

```npm it```

# SYNOPSIS

**npm** **install-test**

# DESCRIPTION

**npm install-test** runs npm install followed by npm test. Combines dependency installation with test execution. Alias: npm it.
