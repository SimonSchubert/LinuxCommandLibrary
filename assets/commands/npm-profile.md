# TAGLINE

manages npm registry user profile

# TLDR

**Show profile**

```npm profile get```

**Set profile field**

```npm profile set [field] [value]```

**Enable 2FA**

```npm profile enable-2fa```

**Disable 2FA**

```npm profile disable-2fa```

# SYNOPSIS

**npm** **profile** _command_ [_options_]

# PARAMETERS

**get** _field_
> Get profile value.

**set** _field_ _value_
> Set profile value.

**enable-2fa** _mode_
> Enable 2FA (auth-only, auth-and-writes).

**disable-2fa**
> Disable 2FA.

# DESCRIPTION

**npm profile** manages npm registry user profile. View and update profile settings. Configure two-factor authentication for account security.
