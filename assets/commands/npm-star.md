# TLDR

**Star a package**

```npm star [package]```

**Star multiple packages**

```npm star [pkg1] [pkg2]```

# SYNOPSIS

**npm** **star** _package_...

# DESCRIPTION

**npm star** marks a package as favorite on npm registry. Shows appreciation for packages you use. Requires npm authentication.
