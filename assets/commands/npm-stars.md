# TLDR

**List starred packages**

```npm stars```

**List user's stars**

```npm stars [username]```

# SYNOPSIS

**npm** **stars** [_user_]

# DESCRIPTION

**npm stars** displays packages starred by a user. Shows your starred packages by default. View another user's starred packages by providing username.
