# TLDR

**Stop application**

```npm stop```

# SYNOPSIS

**npm** **stop** [_args_...]

# DESCRIPTION

**npm stop** runs the stop script defined in package.json. Executes prestop and poststop lifecycle scripts. Used to stop running server processes.
