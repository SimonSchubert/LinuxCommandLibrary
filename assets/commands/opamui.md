# TAGLINE

TUI for browsing and searching OCaml opam packages

# TLDR

**Browse opam packages**

```opamui```

# SYNOPSIS

**opamui** [_options_]

# DESCRIPTION

**opamui** is a TUI for browsing and searching OCaml opam packages in the terminal. It lets you filter packages by name, see which are installed, and inspect package metadata and versions.

# CAVEATS

Requires opam to be installed and configured.

**opamui** was created by **Nicolas Lamirault** (nlamirault) and is written in **OCaml**.
