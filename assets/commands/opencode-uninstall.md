# TAGLINE

Uninstall OpenCode

# TLDR

**Uninstall** OpenCode and remove all related files

```opencode uninstall```

# SYNOPSIS

**opencode uninstall**

# DESCRIPTION

**opencode uninstall** removes the OpenCode installation and all related files from the system, including configuration, cached data, and session history. This provides a clean removal without leaving behind orphaned files.

# CAVEATS

This operation is **irreversible**. All local session data, configuration, and cached credentials will be permanently deleted.
