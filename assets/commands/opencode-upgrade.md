# TAGLINE

Update OpenCode to the latest version

# TLDR

**Upgrade** OpenCode to the latest version

```opencode upgrade```

# SYNOPSIS

**opencode upgrade**

# DESCRIPTION

**opencode upgrade** updates the OpenCode installation to the latest available version. It downloads and replaces the current binary with the newest release, preserving existing configuration and session data.

# CAVEATS

Requires internet access to download the latest release. May require elevated permissions depending on the installation location of the OpenCode binary.
