# TAGLINE

video Editor

# TLDR

**Start OpenShot**

```openshot-qt```

**Open project file**

```openshot-qt [project.osp]```

**Import file**

```openshot-qt [video.mp4]```

# SYNOPSIS

**openshot-qt** [_options_] [_file_]

# PARAMETERS

_FILE_
> Project or media file.

**-h**, **--help**
> Display help information.

**--version**
> Display version.

# DESCRIPTION

**openshot-qt** is OpenShot Video Editor. Non-linear video editing software.

The application provides video editing with timeline. Supports many formats.

# CAVEATS

GUI application. Requires display. Resource intensive for complex projects.

# HISTORY

OpenShot was created as a **free, open-source video editor** for Linux, macOS, and Windows.
