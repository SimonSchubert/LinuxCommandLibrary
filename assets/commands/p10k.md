# TLDR

**Run configuration wizard**

```p10k configure```

**Reload configuration**

```source ~/.p10k.zsh```

**View instant prompt status**

```p10k display```

**Rebuild gitstatus cache**

```p10k reload```

# SYNOPSIS

**p10k** [_command_]

# PARAMETERS

**configure**
> Run setup wizard.

**reload**
> Reload theme.

**display**
> Show prompt segments.

**segment**
> Manage segments.

**help**
> Display help information.

# DESCRIPTION

**p10k** is Powerlevel10k configuration tool. Customizes Zsh prompt theme.

The tool provides interactive setup. Part of Powerlevel10k theme.

p10k configures prompt theme.

# CAVEATS

Requires Powerlevel10k. Zsh only. Nerd Fonts recommended.

# HISTORY

p10k is part of **Powerlevel10k**, a fast Zsh prompt theme.
