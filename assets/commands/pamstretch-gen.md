# TAGLINE

scales images to exact dimensions using pamstretch for integer scaling

# TLDR

**Scale image to exact dimensions**

```pamstretch-gen [800] [600] [input.pam] > [output.pam]```

# SYNOPSIS

**pamstretch-gen** _width_ _height_ [_file_]

# PARAMETERS

_width_
> Target width in pixels.

_height_
> Target height in pixels.

# DESCRIPTION

**pamstretch-gen** scales images to exact dimensions using pamstretch for integer scaling and pamscale for final adjustment. Provides good quality upscaling. Part of Netpbm toolkit.
