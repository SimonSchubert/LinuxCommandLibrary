# TLDR

**Convert PAM to SRF format**

```pamtosrf [input.pam] > [output.srf]```

# SYNOPSIS

**pamtosrf** [_file_]

# DESCRIPTION

**pamtosrf** converts PAM images to SRF (Garmin vehicle image) format used for custom vehicle icons on Garmin GPS devices. Part of Netpbm toolkit.
