# TAGLINE

tunneling tool

# TLDR

**Start Pangolin**

```pangolin```

**Connect to server**

```pangolin [server_url]```

**With configuration**

```pangolin --config [config.yaml]```

# SYNOPSIS

**pangolin** [_options_] [_server_]

# PARAMETERS

_SERVER_
> Server URL to connect.

**--config** _FILE_
> Configuration file.

**--help**
> Display help information.

# DESCRIPTION

**pangolin** is a tunneling tool. Provides secure connections through firewalls.

The tool creates encrypted tunnels. Network access through restrictive networks.

# CAVEATS

Requires server component. Network dependent. Security configuration needed.

# HISTORY

Pangolin was created for **secure tunneling** through restrictive networks.
