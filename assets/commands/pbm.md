# TAGLINE

portable Bitmap format

# TLDR

**Convert to PBM**

```anytopnm [input] | ppmtopgm | pgmtopbm > [output.pbm]```

**View PBM info**

```pnmfile [image.pbm]```

**Convert PBM to ASCII**

```pbmtoascii [image.pbm]```

# SYNOPSIS

Various **pbm*** tools for bitmap manipulation.

# PARAMETERS

PBM format is 1-bit (black and white).

Part of Netpbm toolkit.

# DESCRIPTION

**PBM** is Portable Bitmap format. Binary 1-bit image format.

The format stores black and white images. Part of Netpbm suite.

# CAVEATS

Part of Netpbm. 1-bit only. Smallest Netpbm format.

# HISTORY

PBM was created as part of **Netpbm** for simple bitmap images.
