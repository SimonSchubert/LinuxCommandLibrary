# TAGLINE

converts PBM bitmaps to GraphOn terminal graphics format

# TLDR

**Convert PBM to GraphOn format**

```pbmtogo [input.pbm] > [output.go]```

# SYNOPSIS

**pbmtogo** [_file_]

# DESCRIPTION

**pbmtogo** converts PBM bitmaps to GraphOn terminal graphics format. GraphOn made terminal emulation and graphics software. Part of Netpbm toolkit.
