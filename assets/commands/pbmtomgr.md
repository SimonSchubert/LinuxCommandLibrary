# TLDR

**Convert PBM to MGR format**

```pbmtomgr [input.pbm] > [output.mgr]```

# SYNOPSIS

**pbmtomgr** [_file_]

# DESCRIPTION

**pbmtomgr** converts PBM bitmaps to MGR window system format. MGR was developed at Bellcore in the 1980s. Part of Netpbm toolkit.
