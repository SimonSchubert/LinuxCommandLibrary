# TAGLINE

converts PBM bitmaps to Unix plotformat

# TLDR

**Convert PBM to Unix plot format**

```pbmtoplot [input.pbm] > [output.plot]```

# SYNOPSIS

**pbmtoplot** [_file_]

# DESCRIPTION

**pbmtoplot** converts PBM bitmaps to Unix plot(5) format. Plot format was used for device-independent graphics output. Part of Netpbm toolkit.
