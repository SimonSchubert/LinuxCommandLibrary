# TLDR

**Convert PBM to Bennet Yee "face" format**

```pbmtoybm [input.pbm] > [output.ybm]```

# SYNOPSIS

**pbmtoybm** [_file_]

# DESCRIPTION

**pbmtoybm** converts PBM bitmaps to Bennet Yee's "face" format used by the YBM face display program. Part of Netpbm toolkit.
