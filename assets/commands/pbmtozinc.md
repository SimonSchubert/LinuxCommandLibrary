# TLDR

**Convert PBM to Zinc bitmap**

```pbmtozinc [input.pbm] > [output.zinc]```

# SYNOPSIS

**pbmtozinc** [_file_]

# DESCRIPTION

**pbmtozinc** converts PBM bitmaps to Zinc Interface Library format for GUI development. Part of Netpbm toolkit.
