# TLDR

**Convert PhotoCD overview to PPM**

```pcdovtoppm [input.pcd] > [output.ppm]```

# SYNOPSIS

**pcdovtoppm** [_file_]

# DESCRIPTION

**pcdovtoppm** extracts the overview (thumbnail) image from a Kodak PhotoCD file and converts to PPM. Part of Netpbm toolkit.
