# TLDR

**Convert PCX to PPM**

```pcxtoppm [input.pcx] > [output.ppm]```

# SYNOPSIS

**pcxtoppm** [_file_]

# DESCRIPTION

**pcxtoppm** converts PCX bitmap images to PPM format. Handles 256-color and 24-bit PCX files. Part of Netpbm toolkit.
