# TAGLINE

PHP Extension Community Library manager

# TLDR

**Install extension**

```pecl install [extension]```

**Search extensions**

```pecl search [query]```

**List installed**

```pecl list```

**Uninstall extension**

```pecl uninstall [extension]```

**Upgrade extension**

```pecl upgrade [extension]```

**Show extension info**

```pecl info [extension]```

# SYNOPSIS

**pecl** [_command_] [_options_] [_package_]

# PARAMETERS

**install** _PACKAGE_
> Install extension.

**uninstall** _PACKAGE_
> Remove extension.

**search** _QUERY_
> Search packages.

**list**
> List installed.

**upgrade** _PACKAGE_
> Upgrade extension.

**info** _PACKAGE_
> Show package info.

# DESCRIPTION

**pecl** is PHP Extension Community Library manager. Installs PHP extensions.

The tool manages compiled extensions. Part of PEAR.

# CAVEATS

Requires PHP development tools. May need root. Compiler required.

# HISTORY

PECL was created for **distributing PHP extensions** separately from core.
