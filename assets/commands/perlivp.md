# TLDR

**Run installation verification**

```perlivp```

**Verbose verification**

```perlivp -v```

**Specific test**

```perlivp -p```

# SYNOPSIS

**perlivp** [_options_]

# PARAMETERS

**-v**
> Verbose output.

**-p**
> Test core modules.

**-a**
> Test all modules.

# DESCRIPTION

**perlivp** verifies Perl installation. Runs diagnostic tests.

The tool checks Perl configuration. Installation validation.

perlivp tests Perl install.

# CAVEATS

Part of Perl. Basic tests only. Run after installation.

# HISTORY

perlivp is part of **Perl core** for installation verification.
