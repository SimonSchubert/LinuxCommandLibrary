# TLDR

**Apply Bentley effect to grayscale image**

```pgmbentley [input.pgm] > [output.pgm]```

# SYNOPSIS

**pgmbentley** [_file_]

# DESCRIPTION

**pgmbentley** applies the "Bentley effect" to PGM images, creating a melted/dripping appearance. Named after computer graphics researcher Jon Bentley. Part of Netpbm toolkit.
