# TAGLINE

converts PGM images to Lisp Machine bitmap format

# TLDR

**Convert PGM to Lisp Machine format**

```pgmtolispm [input.pgm] > [output.lispm]```

# SYNOPSIS

**pgmtolispm** [_file_]

# DESCRIPTION

**pgmtolispm** converts PGM images to Lisp Machine bitmap format. Part of Netpbm toolkit.
