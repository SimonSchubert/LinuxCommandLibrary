# TLDR

**Copy PGM image**

```pgmtopgm [input.pgm] > [output.pgm]```

# SYNOPSIS

**pgmtopgm** [_file_]

# DESCRIPTION

**pgmtopgm** copies PGM images, normalizing format. Functions as a no-op filter in pipelines. Part of Netpbm toolkit.
