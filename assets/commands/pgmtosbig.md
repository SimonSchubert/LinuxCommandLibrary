# TAGLINE

converts PGM images to SBIG format used by astronomical CCD cameras

# TLDR

**Convert PGM to SBIG format**

```pgmtosbig [input.pgm] > [output.sbig]```

# SYNOPSIS

**pgmtosbig** [_file_]

# DESCRIPTION

**pgmtosbig** converts PGM images to SBIG (Santa Barbara Instrument Group) format used by astronomical CCD cameras. Part of Netpbm toolkit.
