# TLDR

**Convert PGM to SBIG ST-4 format**

```pgmtost4 [input.pgm] > [output.st4]```

# SYNOPSIS

**pgmtost4** [_file_]

# DESCRIPTION

**pgmtost4** converts PGM images to SBIG ST-4 autoguider format used by astronomical imaging equipment. Part of Netpbm toolkit.
