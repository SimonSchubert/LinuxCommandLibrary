# TLDR

**Convert Atari Degas PI1 to PPM**

```pi1toppm [input.pi1] > [output.ppm]```

# SYNOPSIS

**pi1toppm** [_file_]

# DESCRIPTION

**pi1toppm** converts Atari Degas PI1 low-resolution color images to PPM. PI1 was used by Degas paint program on Atari ST. Part of Netpbm toolkit.
