# TLDR

**Convert Macintosh PICT to PPM**

```picttoppm [input.pict] > [output.ppm]```

**Verbose output**

```picttoppm -verbose [input.pict] > [output.ppm]```

# SYNOPSIS

**picttoppm** [_options_] [_file_]

# PARAMETERS

**-verbose**
> Print information about PICT file.

**-fullres**
> Output full resolution.

# DESCRIPTION

**picttoppm** converts Macintosh PICT images to PPM. PICT was the native graphics format on classic Mac OS. Part of Netpbm toolkit.
