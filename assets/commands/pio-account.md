# TLDR

**Login to PlatformIO account**

```pio account login```

**Show account info**

```pio account show```

**Logout**

```pio account logout```

**Register new account**

```pio account register```

# SYNOPSIS

**pio account** _command_ [_options_]

# PARAMETERS

**login**
> Login to account.

**logout**
> Logout from account.

**show**
> Show account information.

**register**
> Register new account.

**forgot**
> Request password reset.

**-u**, **--username** _user_
> Account username.

**-p**, **--password** _pass_
> Account password.

# DESCRIPTION

**pio account** manages PlatformIO account authentication. Required for accessing PlatformIO Registry, Teams, and remote development features. Free accounts available.
