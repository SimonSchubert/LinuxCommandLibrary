# TLDR

**Show all settings**

```pio settings get```

**Get specific setting**

```pio settings get [projects_dir]```

**Set setting value**

```pio settings set [projects_dir] [~/pio-projects]```

**Reset to defaults**

```pio settings reset```

# SYNOPSIS

**pio settings** _command_ [_options_]

# PARAMETERS

**get** [_name_]
> Get setting value(s).

**set** _name_ _value_
> Set setting value.

**reset**
> Reset to defaults.

# DESCRIPTION

**pio settings** manages PlatformIO configuration settings. Controls default directories, auto-updates, telemetry, and other global preferences.
