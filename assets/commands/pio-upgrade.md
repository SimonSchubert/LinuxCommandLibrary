# TAGLINE

Upgrade PlatformIO Core to latest version

# TLDR

**Upgrade PlatformIO Core**

```pio upgrade```

**Upgrade to development version**

```pio upgrade --dev```

# SYNOPSIS

**pio upgrade** [_options_]

# PARAMETERS

**--dev**
> Upgrade to development version.

# DESCRIPTION

**pio upgrade** upgrades the PlatformIO Core CLI tool itself. Installs the latest stable release or optionally the development version for testing new features.
