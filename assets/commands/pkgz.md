# TLDR

**Search packages**

```pkgz search [query]```

**Install package**

```pkgz install [package]```

**List installed**

```pkgz list```

**Update packages**

```pkgz update```

# SYNOPSIS

**pkgz** [_command_] [_options_]

# PARAMETERS

**search** _QUERY_
> Search packages.

**install** _PACKAGE_
> Install package.

**list**
> List installed.

**update**
> Update packages.

**--help**
> Display help.

# DESCRIPTION

**pkgz** is a package manager wrapper. Simplified package operations.

The tool abstracts package managers. Cross-distribution compatibility.

pkgz manages packages.

# CAVEATS

Wrapper tool. Underlying package manager required.

# HISTORY

pkgz provides **simplified package management** across distributions.
