# TLDR

**View PNM info**

```pnmfile [image.pnm]```

**Convert to other format**

```pnmtopng [image.pnm] > [image.png]```

**Convert from other format**

```pngtopnm [image.png] > [image.pnm]```

# SYNOPSIS

Various **pnm*** tools for PNM image manipulation.

# PARAMETERS

PNM formats include:
- PBM (bitmap)
- PGM (grayscale)
- PPM (color)

Part of Netpbm toolkit.

# DESCRIPTION

**PNM** is Portable Anymap format. Umbrella for PBM, PGM, PPM.

The format provides simple image storage. Text and binary variants.

PNM handles bitmap images.

# CAVEATS

Part of Netpbm. Simple format. Large file sizes.

# HISTORY

PNM was created as part of **Netpbm** for portable image format.
