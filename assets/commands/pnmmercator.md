# TLDR

**Apply Mercator projection**

```pnmmercator [input.pnm] > [output.pnm]```

**Inverse projection**

```pnmmercator -inverse [input.pnm] > [output.pnm]```

# SYNOPSIS

**pnmmercator** [_options_] [_file_]

# PARAMETERS

**-inverse**
> Convert from Mercator to equirectangular.

# DESCRIPTION

**pnmmercator** converts between equirectangular and Mercator map projections. Useful for world map transformations. Part of Netpbm toolkit.
