# TLDR

**Convert PNM to DDIF format**

```pnmtoddif [input.pnm] > [output.ddif]```

# SYNOPSIS

**pnmtoddif** [_options_] [_file_]

# PARAMETERS

**-resolution** _n_
> Output resolution in DPI.

# DESCRIPTION

**pnmtoddif** converts PNM images to DDIF (Digital Document Interchange Format) used by DEC/HP document systems. Part of Netpbm toolkit.
