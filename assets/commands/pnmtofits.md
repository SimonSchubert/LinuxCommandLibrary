# TLDR

**Convert PNM to FITS format**

```pnmtofits [input.pnm] > [output.fits]```

# SYNOPSIS

**pnmtofits** [_file_]

# DESCRIPTION

**pnmtofits** converts PNM images to FITS (Flexible Image Transport System) format used in astronomy. Part of Netpbm toolkit.
