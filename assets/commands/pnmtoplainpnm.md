# TAGLINE

Convert binary PNM to plain ASCII format

# TLDR

**Convert PNM to plain ASCII format**

```pnmtoplainpnm [input.pnm] > [output.pnm]```

# SYNOPSIS

**pnmtoplainpnm** [_file_]

# DESCRIPTION

**pnmtoplainpnm** converts binary PNM images to plain (ASCII) PNM format. Plain format is human-readable but larger. Part of Netpbm toolkit.
