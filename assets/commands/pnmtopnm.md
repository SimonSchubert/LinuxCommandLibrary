# TLDR

**Copy and normalize PNM image**

```pnmtopnm [input.pnm] > [output.pnm]```

# SYNOPSIS

**pnmtopnm** [_file_]

# DESCRIPTION

**pnmtopnm** copies PNM images while normalizing format. Useful as a filter to ensure proper PNM format. Part of Netpbm toolkit.
