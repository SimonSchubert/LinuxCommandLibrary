# TLDR

**Show debug information**

```poetry debug info```

**Resolve dependencies with debug**

```poetry debug resolve```

# SYNOPSIS

**poetry debug** _command_ [_options_]

# PARAMETERS

**info**
> Show system info.

**resolve**
> Debug dependency resolution.

# DESCRIPTION

**poetry debug** provides debugging information. Shows system and environment details.

The tool helps troubleshoot issues. Displays configuration state.

poetry debug aids troubleshooting.

# CAVEATS

For debugging purposes. Shows environment details.

# HISTORY

poetry debug provides **debugging capabilities** for troubleshooting.
