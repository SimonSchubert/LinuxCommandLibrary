# TLDR

**List available Python versions**

```poetry python list```

**Get Python installation info**

```poetry python info```

# SYNOPSIS

**poetry python** _command_ [_options_]

# PARAMETERS

**list**
> List Python versions.

**info**
> Show Python info.

# DESCRIPTION

**poetry python** manages Python environments. Shows available interpreters.

The tool helps select Python versions. Displays environment details.

poetry python manages interpreters.

# CAVEATS

Requires Python 3.8+. Part of environment management.

# HISTORY

poetry python provides **Python interpreter** management capabilities.
