# TLDR

**Update Poetry**

```poetry self update```

**Add Poetry plugin**

```poetry self add [plugin]```

**Remove Poetry plugin**

```poetry self remove [plugin]```

**Show Poetry info**

```poetry self show```

# SYNOPSIS

**poetry self** _command_ [_options_]

# PARAMETERS

**update**
> Update Poetry itself.

**add** _PKG_
> Add Poetry plugin.

**remove** _PKG_
> Remove plugin.

**show**
> Show installed plugins.

# DESCRIPTION

**poetry self** manages Poetry installation. Updates and configures Poetry itself.

The tool handles plugins. Self-manages the installation.

poetry self updates Poetry.

# CAVEATS

Modifies Poetry installation. Not project dependencies.

# HISTORY

poetry self provides **self-management** capabilities for Poetry.
