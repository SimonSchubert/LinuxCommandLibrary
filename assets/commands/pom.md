# TLDR

**Show phase of moon**

```pom```

**Show moon for specific date**

```pom [YYYYMMDD]```

**Show moon for date range**

```pom [start_date] [end_date]```

# SYNOPSIS

**pom** [_date_] [_date_]

# PARAMETERS

_DATE_
> Date in YYYYMMDD format.

# DESCRIPTION

**pom** displays the phase of the moon. BSD calendar utility.

The tool shows lunar phases. Calculates moon cycle.

pom shows moon phase.

# CAVEATS

BSD utility. Date format is YYYYMMDD.

# HISTORY

pom was created as a **BSD utility** for displaying lunar phases.
