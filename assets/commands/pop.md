# TLDR

**Pop value from stack (Forth)**

```pop```

**Pop in dc calculator**

```echo "1 2 3 R p" | dc```

# SYNOPSIS

**pop** [_options_]

# PARAMETERS

Stack operation command.

# DESCRIPTION

**pop** removes top value from stack. Common in stack-based languages.

The tool is used in Forth, dc, and similar. Stack manipulation primitive.

pop removes stack top.

# CAVEATS

Context-dependent. Part of stack-based languages.

# HISTORY

pop is a fundamental **stack operation** in computer science.
