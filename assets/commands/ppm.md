# TLDR

**PPM format overview**

```ppm```

**Convert to PPM**

```anytopnm [image] | ppmtopgm```

**View PPM info**

```pamfile [image.ppm]```

# SYNOPSIS

**ppm** [_file_]

# PARAMETERS

_FILE_
> PPM image file.

# DESCRIPTION

**ppm** is the Portable Pixmap format. Color image format in Netpbm.

The format stores RGB pixel data. Plain text or binary.

ppm defines color images.

# CAVEATS

Netpbm format family. PPM for color, PGM for grayscale, PBM for bitmap.

# HISTORY

PPM was created as part of **Netpbm** portable image format family.
