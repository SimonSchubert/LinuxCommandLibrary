# TLDR

**Create embossed relief effect**

```ppmrelief [input.ppm] > [output.ppm]```

# SYNOPSIS

**ppmrelief** [_file_]

# PARAMETERS

_FILE_
> Input PPM file.

# DESCRIPTION

**ppmrelief** creates embossed relief effect. Edge detection filter.

The tool produces 3D-like appearance. Part of Netpbm.

ppmrelief embosses images.

# CAVEATS

Simple convolution filter. Part of Netpbm suite.

# HISTORY

ppmrelief was created as part of **Netpbm** for relief effect generation.
