# TLDR

**Convert PPM to Encoder YUV**

```ppmtoeyuv [input.ppm] > [output.eyuv]```

# SYNOPSIS

**ppmtoeyuv** [_file_]

# DESCRIPTION

**ppmtoeyuv** converts PPM images to Encoder YUV format (EYUV) used by MPEG encoders. Part of Netpbm toolkit.
