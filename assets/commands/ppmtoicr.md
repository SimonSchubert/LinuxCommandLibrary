# TAGLINE

Convert PPM to NCSA ICR format

# TLDR

**Convert PPM to NCSA ICR format**

```ppmtoicr [input.ppm] > [output.icr]```

# SYNOPSIS

**ppmtoicr** [_options_] [_file_]

# PARAMETERS

**-windowname** _name_
> Specify window name.

**-expand** _n_
> Expand by factor.

# DESCRIPTION

**ppmtoicr** converts PPM images to NCSA ICR (Image Capture Raster) format for X11 display. Part of Netpbm toolkit.
