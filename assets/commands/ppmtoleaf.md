# TLDR

**Convert PPM to Interleaf format**

```ppmtoleaf [input.ppm] > [output.lf]```

# SYNOPSIS

**ppmtoleaf** [_file_]

# DESCRIPTION

**ppmtoleaf** converts PPM images to Interleaf image format for Interleaf document publishing system. Part of Netpbm toolkit.
