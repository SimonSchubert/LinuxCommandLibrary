# TAGLINE

Convert PPM to Atari Neochrome format

# TLDR

**Convert PPM to Atari Neochrome**

```ppmtoneo [input.ppm] > [output.neo]```

# SYNOPSIS

**ppmtoneo** [_file_]

# DESCRIPTION

**ppmtoneo** converts PPM images to Atari Neochrome format for Atari ST computers. Part of Netpbm toolkit.
