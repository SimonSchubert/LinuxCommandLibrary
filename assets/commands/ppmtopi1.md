# TLDR

**Convert PPM to Atari Degas PI1**

```ppmtopi1 [input.ppm] > [output.pi1]```

# SYNOPSIS

**ppmtopi1** [_file_]

# DESCRIPTION

**ppmtopi1** converts PPM images to Atari Degas PI1 low-resolution color format for Atari ST computers. Part of Netpbm toolkit.
