# TLDR

**Convert PPM to Macintosh PICT**

```ppmtopict [input.ppm] > [output.pict]```

# SYNOPSIS

**ppmtopict** [_file_]

# DESCRIPTION

**ppmtopict** converts PPM images to Macintosh PICT format for classic Mac OS. Part of Netpbm toolkit.
