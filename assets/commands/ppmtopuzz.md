# TAGLINE

Convert PPM to X puzzle image format

# TLDR

**Convert PPM to X puzzle format**

```ppmtopuzz [input.ppm] > [output.puzz]```

# SYNOPSIS

**ppmtopuzz** [_file_]

# DESCRIPTION

**ppmtopuzz** converts PPM images to X puzzle format for the xpuzzle game. Part of Netpbm toolkit.
