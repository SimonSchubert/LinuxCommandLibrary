# TLDR

**Display image in terminal**

```ppmtoterm [input.ppm]```

# SYNOPSIS

**ppmtoterm** [_file_]

# DESCRIPTION

**ppmtoterm** displays PPM images directly in the terminal using ANSI color codes. Creates colored block character output. Part of Netpbm toolkit for terminal graphics.
