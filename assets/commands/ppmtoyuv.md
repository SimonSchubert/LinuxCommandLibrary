# TLDR

**Convert PPM to Abekas YUV format**

```ppmtoyuv [input.ppm] > [output.yuv]```

# SYNOPSIS

**ppmtoyuv** [_file_]

# DESCRIPTION

**ppmtoyuv** converts PPM images to Abekas YUV format for video equipment. Part of Netpbm toolkit.
