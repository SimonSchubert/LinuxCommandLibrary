# TAGLINE

Split PPM into YUV component files

# TLDR

**Split PPM into YUV component files**

```ppmtoyuvsplit [base] [input.ppm]```

# SYNOPSIS

**ppmtoyuvsplit** _basename_ [_file_]

# PARAMETERS

_basename_
> Base name for output files (.Y, .U, .V extensions added).

# DESCRIPTION

**ppmtoyuvsplit** splits a PPM image into separate Y, U, V component files for video processing. Part of Netpbm toolkit.
