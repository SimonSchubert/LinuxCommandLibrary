# TLDR

**Apply TV scan line effect**

```ppmtv [input.ppm] > [output.ppm]```

# SYNOPSIS

**ppmtv** [_file_]

# DESCRIPTION

**ppmtv** applies a CRT television scan line effect to PPM images, simulating the appearance of an interlaced TV display. Part of Netpbm toolkit.
