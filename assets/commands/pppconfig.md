# TLDR

**Configure PPP connection**

```pppconfig```

**Run text-based setup**

```pppconfig```

# SYNOPSIS

**pppconfig** [_options_]

# PARAMETERS

**--help**
> Display help.

# DESCRIPTION

**pppconfig** configures PPP dial-up connections. Interactive setup tool.

The tool creates provider files. Manages modem settings.

pppconfig sets up PPP.

# CAVEATS

Debian-specific. For dial-up networking.

# HISTORY

pppconfig was created for **Debian** PPP connection configuration.
