# TLDR

**Configure PPPoE connection**

```pppoe-setup```

**Run interactive setup**

```pppoe-setup```

# SYNOPSIS

**pppoe-setup** [_options_]

# PARAMETERS

Interactive configuration tool.

# DESCRIPTION

**pppoe-setup** configures PPPoE DSL connection interactively. Setup wizard.

The tool creates pppoe.conf file. Prompts for credentials and settings.

pppoe-setup configures DSL.

# CAVEATS

Interactive only. Creates /etc/ppp/pppoe.conf.

# HISTORY

pppoe-setup provides **interactive PPPoE** configuration for DSL connections.
