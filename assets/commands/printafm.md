# TLDR

**Display AFM font metrics**

```printafm [font.afm]```

**Show specific font info**

```printafm [Times-Roman]```

# SYNOPSIS

**printafm** [_font_]

# PARAMETERS

_FONT_
> AFM font file or name.

# DESCRIPTION

**printafm** displays Adobe Font Metrics information. Font inspection tool.

The tool shows font properties. Part of Ghostscript.

printafm reads AFM files.

# CAVEATS

Requires AFM files. Part of Ghostscript suite.

# HISTORY

printafm is part of **Ghostscript** for font metrics display.
