# TAGLINE

TUI tool for tracking relevant pull requests

# TLDR

**Launch the PR tracker**

```prs```

# SYNOPSIS

**prs** [_options_]

# DESCRIPTION

**prs** is a terminal user interface for tracking pull requests that you find relevant without leaving the terminal. It uses a YAML configuration file to define filters for identifying personally important PRs and provides a quick snapshot view of PR activities.

# HISTORY

**prs** was created by **dhth** and is written in **Go**.
