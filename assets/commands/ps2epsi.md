# TLDR

**Convert PS to EPSI**

```ps2epsi [input.ps] [output.epsi]```

**Convert with preview**

```ps2epsi [input.ps]```

# SYNOPSIS

**ps2epsi** _input_ [_output_]

# PARAMETERS

_INPUT_
> Input PostScript file.

_OUTPUT_
> Output EPSI file.

# DESCRIPTION

**ps2epsi** converts PostScript to EPSI format. Adds preview image.

The tool creates Encapsulated PS with preview. Part of Ghostscript.

ps2epsi adds previews.

# CAVEATS

Part of Ghostscript. Creates bitmap preview.

# HISTORY

ps2epsi is part of **Ghostscript** for EPSI conversion.
