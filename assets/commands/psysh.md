# TLDR

**Start PHP REPL**

```psysh```

**Start with specific config**

```psysh -c [config.php]```

**Execute code and exit**

```psysh -e "[code]"```

**Include file**

```psysh --include [file.php]```

# SYNOPSIS

**psysh** [_options_]

# PARAMETERS

**-c** _FILE_
> Configuration file.

**-e** _CODE_
> Execute code.

**--include** _FILE_
> Include file.

**-v**
> Verbose output.

# DESCRIPTION

**psysh** is PHP interactive shell (REPL). Developer console.

The tool provides runtime debugging. Interactive PHP execution.

psysh runs PHP interactively.

# CAVEATS

Requires PHP. Installed via Composer.

# HISTORY

PsySH was created as a **modern PHP REPL** with advanced features.
