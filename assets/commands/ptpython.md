# TLDR

**Start Python REPL**

```ptpython```

**Start with vi mode**

```ptpython --vi```

**Execute file then REPL**

```ptpython -i [script.py]```

**Start IPython mode**

```ptipython```

# SYNOPSIS

**ptpython** [_options_] [_file_]

# PARAMETERS

_FILE_
> Python file to execute.

**--vi**
> Vi editing mode.

**-i**
> Interactive after file.

**--config-file** _FILE_
> Configuration file.

# DESCRIPTION

**ptpython** is an advanced Python REPL. Enhanced interactive Python.

The tool provides syntax highlighting and completion. Better than default REPL.

ptpython enhances Python.

# CAVEATS

Requires installation. Uses prompt_toolkit.

# HISTORY

ptpython was created as an **enhanced Python REPL** with modern features.
