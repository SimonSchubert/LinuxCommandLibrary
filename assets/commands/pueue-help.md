# TLDR

**Show general help**

```pueue help```

**Show help for command**

```pueue help [command]```

**Show add command help**

```pueue help add```

# SYNOPSIS

**pueue help** [_command_]

# PARAMETERS

_COMMAND_
> Command to get help for.

# DESCRIPTION

**pueue help** displays help information. Shows command usage.

The tool provides documentation. Lists available commands.

pueue help shows usage.

# CAVEATS

Also available via --help flag.

# HISTORY

pueue help provides **built-in documentation** for pueue commands.
