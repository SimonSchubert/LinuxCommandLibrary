# TAGLINE

Display Pulumi environment diagnostics

# TLDR

**Show Pulumi version and environment info**

```pulumi about```

**Show in JSON format**

```pulumi about --json```

# SYNOPSIS

**pulumi** **about** [_options_]

# PARAMETERS

**--json**
> Output as JSON.

**-t**, **--transitive**
> Show transitive dependencies.

# DESCRIPTION

**pulumi about** displays diagnostic information about the Pulumi environment. Shows version, plugins, runtime info, and dependencies. Useful for troubleshooting and support requests.
