# TAGLINE

Open Pulumi web console in browser

# TLDR

**Open Pulumi Console in browser**

```pulumi console```

**Open specific stack in console**

```pulumi console -s [stack]```

# SYNOPSIS

**pulumi** **console** [_options_]

# PARAMETERS

**-s**, **--stack** _name_
> Target stack.

# DESCRIPTION

**pulumi console** opens the Pulumi web console in the default browser. Provides web-based management interface for stacks, resources, and deployment history.
