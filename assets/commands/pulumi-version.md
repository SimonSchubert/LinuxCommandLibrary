# TAGLINE

Display installed Pulumi CLI version

# TLDR

**Show Pulumi version**

```pulumi version```

# SYNOPSIS

**pulumi** **version**

# DESCRIPTION

**pulumi version** displays the installed Pulumi CLI version number. Useful for verifying installation and checking for updates.
