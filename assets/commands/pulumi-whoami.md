# TLDR

**Show current user**

```pulumi whoami```

**Show detailed info**

```pulumi whoami -v```

**Show in JSON format**

```pulumi whoami --json```

# SYNOPSIS

**pulumi** **whoami** [_options_]

# PARAMETERS

**-v**, **--verbose**
> Show detailed information.

**--json**
> Output as JSON.

# DESCRIPTION

**pulumi whoami** displays the currently logged-in user and backend. Shows username, organization, and backend URL. Useful for verifying authentication status.
