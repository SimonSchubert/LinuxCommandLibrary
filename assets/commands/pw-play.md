# TAGLINE

Play audio files through PipeWire

# TLDR

This command is an **alias** of **pw-cat --playback**

View documentation for the original command

```tldr pw-cat```

# SYNOPSIS

**pw-play** [_options_] _file_

# DESCRIPTION

**pw-play** is a convenience alias for **pw-cat --playback**, used to play audio files through PipeWire. See **pw-cat** for full documentation.
