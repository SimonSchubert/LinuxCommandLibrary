# TLDR

**Generate passphrase**

```pwqgen```

**Generate with specific word count**

```pwqgen random=5```

**Generate multiple**

```pwqgen count=5```

# SYNOPSIS

**pwqgen** [_options_]

# PARAMETERS

**random=N**
> Number of random words.

**count=N**
> Number of passphrases.

# DESCRIPTION

**pwqgen** generates quality passphrases. Random word combinations.

The tool creates memorable passwords. Uses dictionary words.

pwqgen makes passphrases.

# CAVEATS

Part of passwdqc. May not be installed by default.

# HISTORY

pwqgen is part of **passwdqc** password quality checking suite.
