# TLDR

**Show PXELINUX configuration options**

```pxelinux-options```

**Display available parameters**

```pxelinux-options --help```

# SYNOPSIS

**pxelinux-options** [_options_]

# PARAMETERS

Documentation reference for PXELINUX configuration.

# DESCRIPTION

**pxelinux-options** documents PXELINUX boot options. Configuration reference.

The tool helps configure PXE booting. Part of SYSLINUX project.

pxelinux-options lists options.

# CAVEATS

Reference tool only. Part of SYSLINUX.

# HISTORY

pxelinux-options is part of **SYSLINUX** for PXE boot configuration.
