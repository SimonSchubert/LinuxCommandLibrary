# TLDR

**List installed Python 3 versions**

```py3versions -i```

**Show supported versions**

```py3versions -s```

**Show default version**

```py3versions -d```

**Show all versions**

```py3versions -v```

# SYNOPSIS

**py3versions** [_options_]

# PARAMETERS

**-i**
> Installed versions.

**-s**
> Supported versions.

**-d**
> Default version.

**-v**
> All versions info.

**-r**
> Requested versions.

# DESCRIPTION

**py3versions** lists Python 3 interpreter versions. Debian utility.

The tool shows available Python versions. Package management helper.

py3versions lists Python 3.

# CAVEATS

Debian/Ubuntu specific. Part of python3-minimal.

# HISTORY

py3versions is a **Debian utility** for Python 3 version management.
