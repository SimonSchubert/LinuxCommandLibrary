# TLDR

**Register Python modules (deprecated)**

```pycentral [action] [package]```

# SYNOPSIS

**pycentral** _action_ [_options_]

# PARAMETERS

_ACTION_
> Action to perform.

# DESCRIPTION

**pycentral** managed Python modules. Debian packaging tool (deprecated).

The tool handled module installation. Replaced by dh-python.

pycentral is deprecated.

# CAVEATS

Deprecated. Use dh-python instead.

# HISTORY

pycentral was a **deprecated Debian** tool for Python module management.
