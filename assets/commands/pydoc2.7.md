# TLDR

**View Python 2.7 documentation**

```pydoc2.7 [module]```

**Search documentation**

```pydoc2.7 -k [keyword]```

**Start documentation server**

```pydoc2.7 -p [8080]```

# SYNOPSIS

**pydoc2.7** [_options_] [_module_]

# PARAMETERS

_MODULE_
> Module to document.

**-k** _KEYWORD_
> Search keyword.

**-p** _PORT_
> Start HTTP server.

**-w**
> Write HTML file.

# DESCRIPTION

**pydoc2.7** is pydoc for Python 2.7. Documentation viewer.

The tool shows Python 2 docs. Version-specific pydoc.

pydoc2.7 documents Python 2.

# CAVEATS

Python 2 deprecated. Use pydoc3 instead.

# HISTORY

pydoc2.7 is **Python 2.7's** documentation tool.
