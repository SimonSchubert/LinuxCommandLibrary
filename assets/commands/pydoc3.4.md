# TLDR

**View Python 3.4 documentation**

```pydoc3.4 [module]```

**Search documentation**

```pydoc3.4 -k [keyword]```

**Start documentation server**

```pydoc3.4 -p [8080]```

# SYNOPSIS

**pydoc3.4** [_options_] [_module_]

# PARAMETERS

_MODULE_
> Module to document.

**-k** _KEYWORD_
> Search keyword.

**-p** _PORT_
> Start HTTP server.

**-w**
> Write HTML file.

# DESCRIPTION

**pydoc3.4** is pydoc for Python 3.4. Documentation viewer.

The tool shows Python 3.4 docs. Version-specific pydoc.

pydoc3.4 documents Python 3.4.

# CAVEATS

Python 3.4 specific. Use latest pydoc3.

# HISTORY

pydoc3.4 is **Python 3.4's** documentation tool.
