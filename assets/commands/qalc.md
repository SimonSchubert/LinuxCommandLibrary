# TLDR

**Calculate expression**

```qalc "[2 + 2]"```

**Convert units**

```qalc "[5 kilometers to miles]"```

**Interactive mode**

```qalc```

**Currency conversion**

```qalc "[100 USD to EUR]"```

**Scientific calculation**

```qalc "[sin(45 degrees)]"```

# SYNOPSIS

**qalc** [_options_] [_expression_]

# PARAMETERS

_EXPRESSION_
> Mathematical expression.

**-t**
> Terse output.

**-f** _FILE_
> Read from file.

**-u** _MODE_
> Update exchange rates.

# DESCRIPTION

**qalc** is powerful command-line calculator. Unit conversion and math.

The tool handles complex calculations. Supports physical units.

qalc calculates expressions.

# CAVEATS

Requires libqalculate. Supports currency exchange rates.

# HISTORY

Qalculate was created as a **multi-purpose** calculator with unit support.
