# TLDR

This command is an alias of **qm guest cmd**

View documentation for the original command

```tldr qm guest```

# SYNOPSIS

**qm agent** _vmid_ _command_

# DESCRIPTION

**qm agent** is an alias for **qm guest cmd** that executes commands through the QEMU guest agent. It allows communication with virtual machines via the guest agent installed inside the VM.
