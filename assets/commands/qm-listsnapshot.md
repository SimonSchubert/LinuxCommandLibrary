# TLDR

List **all snapshots** of a virtual machine

```qm listsnapshot 100```

# SYNOPSIS

**qm listsnapshot** _vmid_

# DESCRIPTION

**qm listsnapshot** lists all snapshots for a specific virtual machine in Proxmox VE. It displays snapshot names, creation times, and descriptions.

This helps identify available restore points and manage snapshot storage.
