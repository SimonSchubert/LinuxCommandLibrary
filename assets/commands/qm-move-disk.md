# TAGLINE

Move Proxmox VM disk between storages

# TLDR

This command is an alias of **qm disk move**

View documentation for the original command

```tldr qm disk```

# SYNOPSIS

**qm move-disk** _vmid_ _disk_ _storage_ [_options_]

# DESCRIPTION

**qm move-disk** is an alternative form of **qm disk move** that moves virtual machine disks between storage backends. The hyphenated variant is provided for compatibility.
