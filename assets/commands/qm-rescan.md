# TAGLINE

Rescan Proxmox VM storage volumes

# TLDR

This command is an alias of **qm disk rescan**

View documentation for the original command

```tldr qm disk```

# SYNOPSIS

**qm rescan** [_options_]

# DESCRIPTION

**qm rescan** is an alias for **qm disk rescan** that rescans storage backends and updates disk information for virtual machines. It detects changes in disk sizes and identifies unused disk images.
