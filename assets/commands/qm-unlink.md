# TLDR

This command is an alias of **qm disk unlink**

View documentation for the original command

```tldr qm disk```

# SYNOPSIS

**qm unlink** _vmid_ _disk_

# DESCRIPTION

**qm unlink** is an alias for **qm disk unlink** that removes a disk from a virtual machine's configuration without deleting the underlying storage. The disk becomes unused but remains available for reattachment.
