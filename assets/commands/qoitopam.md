# TLDR

**Convert QOI to PAM format**

```qoitopam [input.qoi] > [output.pam]```

# SYNOPSIS

**qoitopam** [_file_]

# DESCRIPTION

**qoitopam** converts QOI (Quite OK Image) format to PAM. QOI is a fast lossless image format. Part of Netpbm toolkit.
