# TLDR

**Start QOwnNotes**

```qownnotes```

**Open specific folder**

```qownnotes --folder [/path/to/notes]```

**Portable mode**

```qownnotes --portable```

# SYNOPSIS

**qownnotes** [_options_]

# PARAMETERS

**--folder** _PATH_
> Notes folder.

**--portable**
> Portable mode.

**--help**
> Show help.

# DESCRIPTION

**qownnotes** is plain-text note taking application. Markdown notes with Nextcloud sync.

The tool manages markdown notes. Syncs via ownCloud/Nextcloud.

qownnotes takes notes.

# CAVEATS

Qt-based application. Optional cloud sync.

# HISTORY

QOwnNotes was created for **plain-text note** taking with cloud sync.
