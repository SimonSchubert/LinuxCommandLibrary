# TLDR

**Convert QRT ray tracer output to PPM**

```qrttoppm [input.qrt] > [output.ppm]```

# SYNOPSIS

**qrttoppm** [_file_]

# DESCRIPTION

**qrttoppm** converts QRT ray tracer format to PPM. QRT was an early ray tracing program. Part of Netpbm toolkit.
