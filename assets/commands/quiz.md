# TLDR

**Start quiz game**

```quiz```

**Quiz with specific categories**

```quiz [category1] [category2]```

**List available categories**

```quiz -l```

# SYNOPSIS

**quiz** [_options_] [_categories_]

# PARAMETERS

_CATEGORIES_
> Quiz categories.

**-l**
> List categories.

**-t**
> Show time for each.

# DESCRIPTION

**quiz** is educational quiz game. Tests knowledge.

The tool asks questions. BSD games utility.

quiz tests knowledge.
