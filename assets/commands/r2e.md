# TLDR

**Convert RSS to email**

```r2e run```

**Add feed**

```r2e add [name] [url] [email]```

**List feeds**

```r2e list```

**Delete feed**

```r2e delete [name]```

**Create config**

```r2e new [email]```

# SYNOPSIS

**r2e** _command_ [_options_]

# PARAMETERS

**run**
> Process feeds.

**add** _NAME_ _URL_ _EMAIL_
> Add feed.

**list**
> List feeds.

**delete** _NAME_
> Remove feed.

**new** _EMAIL_
> Initialize config.

# DESCRIPTION

**r2e** converts RSS feeds to email. Feed to email bridge.

The tool sends feed updates via email. RSS aggregation.

r2e emails RSS.

# CAVEATS

Requires email setup. Python package.

# HISTORY

rss2email was created for **RSS to email** delivery.
