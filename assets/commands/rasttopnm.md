# TLDR

**Convert Sun raster to PNM**

```rasttopnm [input.ras] > [output.pnm]```

# SYNOPSIS

**rasttopnm** [_file_]

# DESCRIPTION

**rasttopnm** converts Sun Rasterfile images to PNM format. Sun raster was the native format for SunOS/Solaris systems. Part of Netpbm toolkit.
