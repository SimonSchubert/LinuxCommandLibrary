# TLDR

**Start Metro bundler**

```react-native start```

**Start on specific port**

```react-native start --port [8082]```

**Start and reset cache**

```react-native start --reset-cache```

**Start with verbose output**

```react-native start --verbose```

# SYNOPSIS

**react-native start** [_options_]

# PARAMETERS

**--port** _port_
> Server port (default 8081).

**--reset-cache**
> Clear bundler cache.

**--verbose**
> Verbose output.

**--host** _host_
> Server hostname.

**--https**
> Enable HTTPS.

**--config** _path_
> Metro config file.

# DESCRIPTION

**react-native start** starts the Metro JavaScript bundler for React Native development. Serves JavaScript bundle to iOS and Android apps during development. Enables hot reloading and debugging.
