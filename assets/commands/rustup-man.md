# TLDR

**Open man page**

```rustup man [command]```

**Open cargo man page**

```rustup man cargo```

**Open rustc man page**

```rustup man rustc```

# SYNOPSIS

**rustup** **man** [_options_] _command_

# PARAMETERS

**--toolchain** _name_
> Use specific toolchain.

# DESCRIPTION

**rustup man** opens manual pages for Rust tools. Provides access to documentation for cargo, rustc, and other tools. Requires man pages component.
