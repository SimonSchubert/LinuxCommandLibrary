# TLDR

**Convert SBIG to PGM**

```sbigtopgm [input.sbig] > [output.pgm]```

# SYNOPSIS

**sbigtopgm** [_file_]

# DESCRIPTION

**sbigtopgm** converts SBIG (Santa Barbara Instrument Group) astronomical CCD camera images to PGM format. Part of Netpbm toolkit.
