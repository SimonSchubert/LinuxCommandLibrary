# TLDR

**Convert warts to pcap format**

```sc_warts2pcap [input.warts] > [output.pcap]```

# SYNOPSIS

**sc_warts2pcap** [_options_] _file_

# DESCRIPTION

**sc_warts2pcap** converts scamper warts files to pcap format for analysis with tools like Wireshark or tcpdump. Part of scamper toolkit.
