# TLDR

**Concatenate warts files**

```sc_wartscat [file1.warts] [file2.warts] > [combined.warts]```

# SYNOPSIS

**sc_wartscat** _files_...

# DESCRIPTION

**sc_wartscat** concatenates multiple scamper warts files into a single file. Part of scamper toolkit.
