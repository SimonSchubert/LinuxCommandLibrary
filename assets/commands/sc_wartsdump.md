# TAGLINE

Display raw contents of scamper warts files

# TLDR

**Display raw warts file contents**

```sc_wartsdump [input.warts]```

# SYNOPSIS

**sc_wartsdump** [_options_] _file_

# PARAMETERS

**-d**
> Debug mode with extra detail.

# DESCRIPTION

**sc_wartsdump** displays the raw contents of scamper warts files for debugging or inspection. Part of scamper toolkit.
