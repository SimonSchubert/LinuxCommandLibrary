# TAGLINE

Repair corrupted scamper warts files

# TLDR

**Repair corrupted warts file**

```sc_wartsfix [input.warts] > [fixed.warts]```

# SYNOPSIS

**sc_wartsfix** _file_

# DESCRIPTION

**sc_wartsfix** attempts to repair truncated or corrupted scamper warts files by recovering valid records. Part of scamper toolkit.
