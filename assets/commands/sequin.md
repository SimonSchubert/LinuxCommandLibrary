# TLDR

**Start consumer**

```sequin start```

**List streams**

```sequin streams```

**Create consumer**

```sequin consumers create [name]```

**Check status**

```sequin status```

**View logs**

```sequin logs```

# SYNOPSIS

**sequin** _command_ [_options_]

# PARAMETERS

**start**
> Start consumer.

**streams**
> List streams.

**consumers**
> Manage consumers.

**status**
> Show status.

**logs**
> View logs.

**--config** _FILE_
> Config file.

# DESCRIPTION

**sequin** streams database changes. It captures CDC events.

Postgres CDC tool. Change data capture.

Real-time streaming. Database to queue.

Consumer management. Process changes.

HTTP webhooks supported. Flexible delivery.

# CAVEATS

Postgres focused. Service connection needed. Configuration required.

# HISTORY

**Sequin** is a change data capture tool for streaming database changes in real-time.
