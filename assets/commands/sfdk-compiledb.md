# TLDR

**Generate compilation database**

```sfdk compiledb```

# SYNOPSIS

**sfdk compiledb** [_options_]

# DESCRIPTION

**sfdk compiledb** generates a compile_commands.json compilation database for IDE integration and tooling. Part of Sailfish SDK.
