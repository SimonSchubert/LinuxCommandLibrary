# TAGLINE

Manage Sailfish SDK configuration settings

# TLDR

**Show current configuration**

```sfdk config```

**Set target device**

```sfdk config --device [device_name]```

**Set build target**

```sfdk config --target [target_name]```

# SYNOPSIS

**sfdk config** [_options_]

# PARAMETERS

**--device** _name_
> Set target device.

**--target** _name_
> Set build target.

**--specfile** _file_
> Set spec file.

# DESCRIPTION

**sfdk config** manages Sailfish SDK configuration. Sets build targets, devices, and project options. Part of Sailfish SDK.
