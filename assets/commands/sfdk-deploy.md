# TAGLINE

Deploy Sailfish apps to devices or emulators

# TLDR

**Deploy application to device**

```sfdk deploy```

**Deploy with specific method**

```sfdk deploy --sdk```

# SYNOPSIS

**sfdk deploy** [_options_]

# PARAMETERS

**--sdk**
> Deploy via SDK connection.

**--manual**
> Manual deployment.

**--rsync**
> Use rsync for deployment.

# DESCRIPTION

**sfdk deploy** deploys built Sailfish OS applications to devices or emulators. Part of Sailfish SDK.
