# TLDR

**List available device models**

```sfdk emulator device-model list```

# SYNOPSIS

**sfdk emulator device-model list** [_options_]

# DESCRIPTION

**sfdk emulator device-model list** shows available device models for Sailfish OS emulator. Part of Sailfish SDK.
