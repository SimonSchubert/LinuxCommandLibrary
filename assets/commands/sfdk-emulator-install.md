# TLDR

**Install emulator**

```sfdk emulator install [emulator_name]```

# SYNOPSIS

**sfdk emulator install** _name_ [_options_]

# DESCRIPTION

**sfdk emulator install** installs a new Sailfish OS emulator image. Part of Sailfish SDK.
