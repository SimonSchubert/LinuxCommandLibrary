# TLDR

**Remove emulator**

```sfdk emulator remove [emulator_name]```

# SYNOPSIS

**sfdk emulator remove** _name_ [_options_]

# DESCRIPTION

**sfdk emulator remove** removes an installed Sailfish OS emulator. Part of Sailfish SDK.
