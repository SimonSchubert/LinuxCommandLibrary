# TLDR

**Start emulator**

```sfdk emulator start```

**Stop emulator**

```sfdk emulator stop```

**List emulators**

```sfdk emulator list```

# SYNOPSIS

**sfdk emulator** _command_ [_options_]

# PARAMETERS

**start**
> Start emulator.

**stop**
> Stop emulator.

**list**
> List emulators.

# DESCRIPTION

**sfdk emulator** manages Sailfish OS emulator instances for testing applications. Part of Sailfish SDK.
