# TAGLINE

Initialize new Sailfish OS projects

# TLDR

**Initialize new project**

```sfdk init```

**Initialize with template**

```sfdk init --template [qml-app]```

# SYNOPSIS

**sfdk init** [_options_]

# PARAMETERS

**--template** _name_
> Project template.

**--name** _name_
> Project name.

# DESCRIPTION

**sfdk init** initializes a new Sailfish OS project from templates. Part of Sailfish SDK.
