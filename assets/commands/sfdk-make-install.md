# TAGLINE

Install built files in Sailfish environment

# TLDR

**Install built files**

```sfdk make-install```

**Install to custom prefix**

```sfdk make-install DESTDIR=[path]```

# SYNOPSIS

**sfdk make-install** [_options_]

# DESCRIPTION

**sfdk make-install** runs make install in the Sailfish OS build environment. Part of Sailfish SDK.
