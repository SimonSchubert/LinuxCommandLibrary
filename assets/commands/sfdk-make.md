# TAGLINE

Run make in Sailfish build environment

# TLDR

**Build project with make**

```sfdk make```

**Build with parallel jobs**

```sfdk make -j[4]```

**Clean build**

```sfdk make clean```

# SYNOPSIS

**sfdk make** [_make_options_]

# DESCRIPTION

**sfdk make** runs make inside the Sailfish OS build environment. Part of Sailfish SDK.
