# TLDR

**Prepare sources for build**

```sfdk prepare```

# SYNOPSIS

**sfdk prepare** [_options_]

# DESCRIPTION

**sfdk prepare** prepares source files for building, applying patches and generating files as needed. Part of Sailfish SDK.
