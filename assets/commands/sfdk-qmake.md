# TLDR

**Run qmake in build environment**

```sfdk qmake```

**Configure with options**

```sfdk qmake CONFIG+=debug```

# SYNOPSIS

**sfdk qmake** [_qmake_options_]

# DESCRIPTION

**sfdk qmake** runs qmake inside the Sailfish OS build environment with correct Qt configuration. Part of Sailfish SDK.
