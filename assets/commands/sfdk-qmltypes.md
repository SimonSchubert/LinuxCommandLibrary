# TLDR

**Generate QML type information**

```sfdk qmltypes```

# SYNOPSIS

**sfdk qmltypes** [_options_]

# DESCRIPTION

**sfdk qmltypes** generates QML type information for IDE integration and type checking. Part of Sailfish SDK.
