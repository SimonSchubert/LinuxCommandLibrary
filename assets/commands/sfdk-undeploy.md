# TAGLINE

Remove deployed apps from devices

# TLDR

**Remove deployed application**

```sfdk undeploy```

# SYNOPSIS

**sfdk undeploy** [_options_]

# DESCRIPTION

**sfdk undeploy** removes a previously deployed application from the target device. Part of Sailfish SDK.
