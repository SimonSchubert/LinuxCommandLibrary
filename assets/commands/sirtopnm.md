# TLDR

**Convert Solitaire image to PNM**

```sirtopnm [input.sir] > [output.pnm]```

# SYNOPSIS

**sirtopnm** [_file_]

# DESCRIPTION

**sirtopnm** converts Solitaire image recorder format to PNM. Solitaire was a film recorder format. Part of Netpbm toolkit.
