# TLDR

**Convert SPOT satellite image to PGM**

```spottopgm [input.spot] > [output.pgm]```

# SYNOPSIS

**spottopgm** [_file_]

# DESCRIPTION

**spottopgm** converts SPOT satellite image format to PGM. SPOT satellites capture Earth observation imagery. Part of Netpbm toolkit.
