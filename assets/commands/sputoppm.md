# TLDR

**Convert Atari Spectrum 512 to PPM**

```sputoppm [input.spu] > [output.ppm]```

# SYNOPSIS

**sputoppm** [_file_]

# DESCRIPTION

**sputoppm** converts Atari Spectrum 512 uncompressed format to PPM. Spectrum 512 displayed 512 colors on Atari ST. Part of Netpbm toolkit.
