# TLDR

**Convert Garmin SRF to PAM**

```srftopam [input.srf] > [output.pam]```

# SYNOPSIS

**srftopam** [_file_]

# DESCRIPTION

**srftopam** converts Garmin SRF (vehicle image) format to PAM. SRF is used for custom vehicle icons on Garmin GPS devices. Part of Netpbm toolkit.
