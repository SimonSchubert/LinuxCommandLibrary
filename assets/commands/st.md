# TLDR

This command refers to multiple tools. See documentation for:

**st (terminal emulator)**

```tldr st.1```

**st (statistics tool)**

```tldr st.2```

# DESCRIPTION

The command **st** can refer to different programs depending on context:

1. **st** - Simple terminal emulator for X11 from suckless.org
2. **st** - Statistics tool for command-line data analysis

See the specific documentation for each variant.
