# TLDR

**Convert SBIG ST-4 to PGM**

```st4topgm [input.st4] > [output.pgm]```

# SYNOPSIS

**st4topgm** [_file_]

# DESCRIPTION

**st4topgm** converts SBIG ST-4 autoguider format to PGM. ST-4 was an astronomical CCD autoguider. Part of Netpbm toolkit.
