# TAGLINE

Display SteamOS system diagnostics

# TLDR

View SteamOS **system information**

```sudo steamos-dump-info```

# SYNOPSIS

**steamos-dump-info**

# DESCRIPTION

**steamos-dump-info** displays detailed system information about a SteamOS installation. It collects and outputs various system details useful for troubleshooting and support.

# PARAMETERS

This command takes no parameters.

# CAVEATS

Requires root privileges. This command is specific to SteamOS (used on Steam Deck and other Valve devices). The output includes hardware, software, and configuration details.

# HISTORY

**steamos-dump-info** is part of the SteamOS system tools developed by Valve for the Steam Deck and other SteamOS-based devices.
