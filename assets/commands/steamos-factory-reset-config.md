# TLDR

Generate factory reset **configuration**

```steamos-factory-reset-config```

# SYNOPSIS

**steamos-factory-reset-config**

# DESCRIPTION

**steamos-factory-reset-config** generates the configuration files required by `steamos-factory-reset`. This command is typically called internally by the factory reset process and is not meant to be run manually.

The generated configuration defines what data to preserve and what to erase during the factory reset process.

# CAVEATS

Internal command not intended for manual use. Running this command directly may have unintended consequences. Part of the factory reset infrastructure.

# HISTORY

**steamos-factory-reset-config** is part of the **SteamOS** utilities developed by **Valve** for Steam Deck system recovery operations.
