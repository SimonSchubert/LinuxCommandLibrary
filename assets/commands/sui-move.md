# TAGLINE

Sui Move smart contract development tools

# TLDR

**Build Move package**

```sui move build```

**Run tests**

```sui move test```

**Publish package**

```sui move publish --gas-budget [10000]```

**Create new package**

```sui move new [package_name]```

# SYNOPSIS

**sui move** _command_ [_options_]

# PARAMETERS

**build**
> Build Move package.

**test**
> Run Move tests.

**publish**
> Publish to network.

**new** _name_
> Create new package.

**prove**
> Run Move Prover.

**--gas-budget** _amount_
> Gas budget for transactions.

**--path** _path_
> Package path.

# DESCRIPTION

**sui move** manages Move smart contracts on Sui blockchain. Build, test, and deploy Move packages. Part of Sui blockchain development toolkit.
