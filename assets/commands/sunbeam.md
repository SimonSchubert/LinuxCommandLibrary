# TAGLINE

General purpose command-line launcher

# TLDR

**Install an extension**

```sunbeam extension install [origin]```

**List installed extensions**

```sunbeam extension list```

**Open a file or URL**

```sunbeam open [target]```

# SYNOPSIS

**sunbeam** [_command_] [_flags_]

# DESCRIPTION

**sunbeam** lets you define UIs composed of a succession of views from simple scripts written in any language. It is a hybrid between application launchers like Raycast and fuzzy-finders like fzf. It supports extensions for GitHub, Nix, Google Search, Hacker News, and integrates with tmux and zellij.

# HISTORY

**sunbeam** was created by **pomdtr** and is written in **Go**.
