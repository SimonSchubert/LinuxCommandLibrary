# TLDR

**Convert Sun icon to PNM**

```sunicontopnm [input.icon] > [output.pnm]```

# SYNOPSIS

**sunicontopnm** [_file_]

# DESCRIPTION

**sunicontopnm** converts Sun icon format to PNM. Sun icons were used on Sun workstations. Part of Netpbm toolkit.
