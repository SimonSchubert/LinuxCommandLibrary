# TAGLINE

Dead simple TUI task timer

# TLDR

**Launch the task timer**

```tt```

# SYNOPSIS

**tt**

# DESCRIPTION

**tasktimer** is a minimal TUI task timer. You type a task description and press Enter to start timing; press Enter again to stop, or type a new task to switch. All task durations are summed in the header. Press Ctrl+C to stop and display a summary.

# HISTORY
