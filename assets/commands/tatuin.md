# TAGLINE

Task aggregator TUI for multiple providers

# TLDR

**Launch the task aggregation TUI**

```tatuin```

**Use a custom configuration file**

```tatuin --config [path/to/config]```

# SYNOPSIS

**tatuin** [_options_]

# DESCRIPTION

**tatuin** aggregates tasks from platforms like Todoist, Obsidian, GitLab, and GitHub into a single terminal interface. It supports creating tasks, filtering, status updates, saveable UI states, and custom theming.

# HISTORY

**tatuin** was created by **Konnov Konstantin** (panter-dsd) and is written in **Rust**.
