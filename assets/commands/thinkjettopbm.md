# TLDR

**Convert HP ThinkJet to PBM**

```thinkjettopbm [input.thinkjet] > [output.pbm]```

# SYNOPSIS

**thinkjettopbm** [_file_]

# DESCRIPTION

**thinkjettopbm** converts HP ThinkJet printer format to PBM. ThinkJet was HP's first inkjet printer line. Part of Netpbm toolkit.
