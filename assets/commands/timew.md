# TLDR

**Start tracking**

```timew start [tag1] [tag2]```

**Stop tracking**

```timew stop```

**Show current**

```timew```

**Show summary**

```timew summary```

**Show week**

```timew summary :week```

**Continue last task**

```timew continue```

**Track past time**

```timew track [9:00] - [12:00] [meeting]```

# SYNOPSIS

**timew** [_command_] [_options_] [_args_]

# PARAMETERS

**start**
> Begin tracking.

**stop**
> End tracking.

**summary**
> Show report.

**continue**
> Resume last.

**track**
> Add past time.

**:day**
> Today filter.

**:week**
> This week.

# DESCRIPTION

**timew** tracks time intervals. It's Timewarrior.

Tag-based tracking. Flexible categorization.

Reporting built-in. Summaries and exports.

Taskwarrior integration. Combined workflow.

Interval tracking. Start/stop or specify.

# CAVEATS

Learning curve for ranges. Data in ~/.timewarrior. Taskwarrior optional.

# HISTORY

**Timewarrior** was created by the **Taskwarrior** team as a companion time tracking tool.
