# TAGLINE

Interactive TeX Live Manager session

# TLDR

**Start interactive shell**

```tlmgr shell```

**Run commands interactively**

```tlmgr shell```

# SYNOPSIS

**tlmgr shell**

# DESCRIPTION

**tlmgr shell** starts an interactive TeX Live Manager session. Execute multiple commands without restarting tlmgr. Faster for batch operations and exploration.
