# TLDR

**Start typing test**

```toipe```

**Use specific wordlist**

```toipe -w [wordlist]```

**Set word count**

```toipe -n [50]```

**Show available wordlists**

```toipe --list```

# SYNOPSIS

**toipe** [_-w wordlist_] [_-n count_] [_options_]

# PARAMETERS

**-w** _WORDLIST_
> Word list to use.

**-n** _COUNT_
> Number of words.

**--list**
> Show wordlists.

**--help**
> Show help.

# DESCRIPTION

**toipe** is a typing test. It runs in the terminal.

Measure typing speed. WPM calculation.

Multiple word lists. Different difficulties.

Clean interface. Distraction-free.

Accuracy tracking. Error counting.

# CAVEATS

Terminal-based only. Rust implementation. Limited customization.

# HISTORY

**toipe** was created as a terminal-based typing test application written in Rust.
