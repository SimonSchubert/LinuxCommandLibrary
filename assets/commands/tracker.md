# TAGLINE

Real-time satellite tracking and orbit prediction in the terminal

# TLDR

**Launch the satellite tracking TUI**

```tracker```

# SYNOPSIS

**tracker** [_options_]

# DESCRIPTION

**tracker** is a terminal-based application for real-time satellite tracking and orbit prediction using the SGP4 model. It fetches and updates orbital elements (TLE data) automatically via the internet, displaying sky views and world map trajectories.

# HISTORY

**tracker** was created by **ShenMian** and is written in **Rust**.
