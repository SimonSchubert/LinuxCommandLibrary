# TLDR

**Start window manager**

```twm```

**With display**

```twm -display [localhost:0]```

**Verbose mode**

```twm -v```

**Single screen**

```twm -s```

# SYNOPSIS

**twm** [_-display dpy_] [_-v_] [_-s_] [_options_]

# PARAMETERS

**-display** _DPY_
> X display to use.

**-v**
> Verbose mode.

**-s**
> Single screen.

**-f** _FILE_
> Config file.

# DESCRIPTION

**twm** is a window manager. It's Tab Window Manager.

Classic X11 WM. Minimal and simple.

Window decorations. Title bars, borders.

Virtual screens. Workspace support.

Highly configurable. .twmrc file.

# CAVEATS

Very minimal. No modern features. Legacy X11.

# HISTORY

**twm** (Tab Window Manager) was the default window manager for X11, created by Tom LaStrange in 1987.
