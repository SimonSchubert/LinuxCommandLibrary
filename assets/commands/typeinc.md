# TLDR

**Start typing**

```typeinc```

**Practice with file**

```typeinc [text.txt]```

**Set duration**

```typeinc -t [60]```

**Custom word count**

```typeinc -w [100]```

# SYNOPSIS

**typeinc** [_-t seconds_] [_-w count_] [_options_] [_file_]

# PARAMETERS

**-t** _SEC_
> Duration in seconds.

**-w** _COUNT_
> Word count.

**-l** _LANG_
> Language.

**--help**
> Show help.

# DESCRIPTION

**typeinc** is a typing trainer. It measures speed.

Speed measurement. WPM calculation.

Accuracy tracking. Error counting.

Custom texts. Practice anything.

Terminal-based. Distraction-free.

# CAVEATS

Terminal only. Limited customization. Practice tool.

# HISTORY

**typeinc** is a terminal typing practice application for improving typing speed and accuracy.
