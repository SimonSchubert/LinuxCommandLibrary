# TAGLINE

Cozy typing speed tester in the terminal

# TLDR

**Start a default typing test**

```typioca```

**Start a 60-second test**

```typioca --time [60]```

**Start a 50-word test**

```typioca --words [50]```

# SYNOPSIS

**typioca** [_options_]

# DESCRIPTION

**typioca** is a minimal, terminal-based typing speed tester that provides accurate Words Per Minute results. It offers time-based or word/sentence count-based tests with multiple word lists sourced from classical books.

# HISTORY

**typioca** was created by **Lukas Bloznelis** (bloznelis) and is written in **Go**.
