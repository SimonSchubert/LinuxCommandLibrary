# TLDR

**Start server**

```var-dump-server```

**Custom port**

```var-dump-server --host=[127.0.0.1:9912]```

**HTML format**

```var-dump-server --format=html```

# SYNOPSIS

**var-dump-server** [_--host address_] [_--format fmt_] [_options_]

# PARAMETERS

**--host** _ADDR_
> Bind address.

**--format** _FMT_
> Output format.

**--help**
> Show help.

# DESCRIPTION

**var-dump-server** receives dumps. It's for Symfony VarDumper.

Debug server. Receive var dumps.

Centralized output. Remote debugging.

Formatted display. Structured output.

Symfony component. PHP debugging.

# CAVEATS

Symfony/PHP specific. VarDumper required. Development tool.

# HISTORY

**var-dump-server** is part of **Symfony VarDumper** component for receiving and displaying variable dumps from PHP applications.
