# TLDR

**Start storage pool**

```virsh pool-start [pool]```

# SYNOPSIS

**virsh** **pool-start** _pool_

# DESCRIPTION

**virsh pool-start** activates a defined storage pool. Makes pool available for creating and using volumes. Pool must be defined before starting.
