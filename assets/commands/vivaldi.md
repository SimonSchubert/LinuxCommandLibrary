# TLDR

**Start browser**

```vivaldi```

**Open URL**

```vivaldi [https://example.com]```

**New window**

```vivaldi --new-window [url]```

**Incognito mode**

```vivaldi --incognito```

**With profile**

```vivaldi --profile-directory="[Profile 1]"```

# SYNOPSIS

**vivaldi** [_--new-window_] [_--incognito_] [_options_] [_url_]

# PARAMETERS

**--new-window**
> New window.

**--incognito**
> Private browsing.

**--profile-directory** _DIR_
> Use profile.

**--app** _URL_
> App mode.

**--help**
> Show help.

# DESCRIPTION

**vivaldi** is a web browser. It's highly customizable.

Chromium-based. Web standards.

Tab management. Stacking, tiling.

Built-in tools. Notes, mail, calendar.

Extensive customization. UI flexibility.

# CAVEATS

Proprietary parts. Chromium base. Resource usage.

# HISTORY

**Vivaldi** was created by **Jon von Tetzchner**, co-founder of Opera, for power users wanting customization.
