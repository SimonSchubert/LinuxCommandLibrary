# TLDR

**Create project from template**

```vue init [template] [project-name]```

**Create from webpack template**

```vue init webpack [project-name]```

**Create from local template**

```vue init ~/my-template [project-name]```

# SYNOPSIS

**vue** **init** [_options_] _template_ _project-name_

# PARAMETERS

**--clone**
> Use git clone instead of download.

**--offline**
> Use cached template.

# DESCRIPTION

**vue init** creates new Vue.js projects from templates. Legacy command from Vue CLI 2.x. For Vue CLI 3+, use vue create instead. Still useful for custom templates.

# CAVEATS

This is a Vue CLI 2.x command. Vue CLI 3+ uses vue create for new projects.
