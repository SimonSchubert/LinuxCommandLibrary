# TAGLINE

TUI for managing Wake-on-LAN devices

# TLDR

**Launch the Wake-on-LAN TUI**

```wakey```

# SYNOPSIS

**wakey**

# DESCRIPTION

**wakey** is a terminal user interface for managing and waking devices using Wake-on-LAN. It lets you organize devices into groups, send magic packets to wake machines, and ping devices to check their status. Press **n** to add a new device with its MAC address. Configuration is stored in **~/.wakey_config.json**.

# HISTORY

**wakey** was created by **Jonathan Ruiz** (jonathanruiz) and is written in **Go**.
