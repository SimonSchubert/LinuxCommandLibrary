# TLDR

Generate with **wal** palette

```wal-telegram```

Use **background** image

```wal-telegram --background=[path/to/image]```

Use **tiled** color background

```wal-telegram --tiled```

Apply **blur** to background

```wal-telegram -g```

Set **destination**

```wal-telegram --destination=[path/to/destination]```

**Restart** Telegram after generation

```wal-telegram --restart```

# SYNOPSIS

**wal-telegram** [_OPTIONS_]

# PARAMETERS

**--background** _PATH_
> Use specified background image

**--tiled**
> Use colored background from palette

**-g**
> Apply gaussian blur to background

**--destination** _PATH_
> Output location for generated theme

**--restart**
> Restart Telegram after generating theme

# DESCRIPTION

**wal-telegram** generates Telegram themes based on colors generated by pywal/wal. It creates custom themes that match your terminal and desktop color scheme.

By default, it uses the current wallpaper set by feh and wal's color palette.

# CAVEATS

Requires pywal/wal to be configured. Wallpaper detection works with feh by default. Theme location may need to be configured in Telegram.
