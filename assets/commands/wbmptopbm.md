# TLDR

**Convert WBMP to PBM**

```wbmptopbm [input.wbmp] > [output.pbm]```

# SYNOPSIS

**wbmptopbm** [_file_]

# DESCRIPTION

**wbmptopbm** converts WBMP (Wireless Bitmap) format to PBM. WBMP was used in WAP mobile web pages. Part of Netpbm toolkit.
