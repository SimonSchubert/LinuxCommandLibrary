# TLDR

**Start typing test**

```wpm```

**Practice with text file**

```wpm [text.txt]```

**Set duration**

```wpm -d [60]```

**Set word count**

```wpm -w [50]```

# SYNOPSIS

**wpm** [_-d seconds_] [_-w count_] [_options_] [_file_]

# PARAMETERS

**-d** _SEC_
> Duration.

**-w** _COUNT_
> Word count.

**--help**
> Show help.

# DESCRIPTION

**wpm** measures typing speed. It's a typing test.

Words per minute. Speed measurement.

Accuracy tracking. Error counting.

Custom texts. Practice anything.

Terminal-based. Distraction-free.

# CAVEATS

Terminal only. Various implementations. Practice tool.

# HISTORY

**wpm** (words per minute) tools measure typing speed in the terminal for practice and benchmarking.
