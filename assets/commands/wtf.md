# TLDR

**Expand** acronym

```wtf [IMO]```

Search **computer** terms

```wtf -t comp [WWW]```

# SYNOPSIS

**wtf** [_OPTIONS_] _ACRONYM_

# PARAMETERS

**-t** _TYPE_
> Specify search type (comp for computer-related)

# DESCRIPTION

**wtf** shows the expansions of acronyms. It looks up acronyms in a local database and displays their meanings.

The tool includes general acronyms and specialized categories like computer terminology.

# CAVEATS

Database may not include all acronyms. Some acronyms have multiple meanings. Primarily Unix/hacker jargon focused.

# HISTORY

**wtf** stands for "What The F..." and is part of the bsdgames package, originating from BSD Unix culture.
