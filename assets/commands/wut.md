# TAGLINE

CLI that explains your last terminal command output

# TLDR

**Explain the output of the last command**

```wut```

# SYNOPSIS

**wut**

# DESCRIPTION

**wut** is a terminal assistant that uses an LLM to explain whatever is currently in your terminal. It can decipher stack traces, error codes, invalid commands, and log summaries. Requires running inside a **tmux** or **screen** session.

# HISTORY

**wut** was created by **Jonathan Shobrook** (shobrook) and is written in **Python**.
