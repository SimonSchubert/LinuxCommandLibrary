# TAGLINE

TUI wrapper for Void Linux package manager

# TLDR

**Launch the package manager TUI**

```xbps-tui```

# SYNOPSIS

**xbps-tui**

# DESCRIPTION

**xbps-tui** is an interactive terminal frontend for the xbps package manager used by Void Linux. It supports fuzzy searching packages by name and description, filtering by installed, updatable, or marked packages, and marking packages for installation or removal with root execution. Press **/** to enter fuzzy search mode.

# HISTORY

**xbps-tui** was created by **lukeflo** and is written in **Rust**.
