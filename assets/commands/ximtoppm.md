# TLDR

**Convert XIM to PPM**

```ximtoppm [input.xim] > [output.ppm]```

# SYNOPSIS

**ximtoppm** [_file_]

# DESCRIPTION

**ximtoppm** converts XIM (X Image) format to PPM. XIM was used by some X11 applications. Part of Netpbm toolkit.
