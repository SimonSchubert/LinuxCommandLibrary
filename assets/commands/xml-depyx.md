# TLDR

**Convert PYX to XML**

```xml depyx [file.pyx]```

**Convert from stdin**

```cat [file.pyx] | xml depyx```

# SYNOPSIS

**xml depyx** [_options_] [_file_]

# PARAMETERS

_file_
> Input PYX file.

# DESCRIPTION

**xml depyx** converts PYX notation back to XML. Part of xmlstarlet toolkit. Reconstructs XML documents from line-oriented PYX format after text processing.
