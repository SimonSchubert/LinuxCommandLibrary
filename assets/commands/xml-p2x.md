# TAGLINE

Convert PYX format back to XML

# TLDR

**Convert PYX to XML**

```xml p2x [file.pyx]```

# SYNOPSIS

**xml p2x** [_options_] [_file_]

# PARAMETERS

_file_
> Input PYX file.

# DESCRIPTION

**xml p2x** converts PYX format back to XML. Part of xmlstarlet toolkit. Alias for xml depyx command.

# INSTALL

```dnf: sudo dnf install xmlstarlet```

```pacman: sudo pacman -S xmlstarlet```

```apk: sudo apk add xmlstarlet```

```zypper: sudo zypper install xmlstarlet```

```brew: brew install xmlstarlet```

```nix: nix profile install nixpkgs#xmlstarlet```
