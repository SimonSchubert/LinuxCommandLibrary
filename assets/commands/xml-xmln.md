# TLDR

**Show namespace information**

```xml xmln [file.xml]```

# SYNOPSIS

**xml xmln** [_options_] [_file_]

# PARAMETERS

_file_
> Input XML file.

# DESCRIPTION

**xml xmln** displays namespace information in XML documents. Part of xmlstarlet toolkit. Shows declared namespaces with their prefixes and URIs.
