# TLDR

**Start xonsh**

```xonsh```

**Run command**

```xonsh -c "[print('hello')]"```

**Run script**

```xonsh [script.xsh]```

**Interactive Python mode**

```xonsh -i```

# SYNOPSIS

**xonsh** [_-c command_] [_-i_] [_options_] [_file_]

# PARAMETERS

**-c** _CMD_
> Execute command.

**-i**
> Interactive mode.

**--login**
> Login shell.

**--help**
> Show help.

# DESCRIPTION

**xonsh** is Python-powered shell. It combines bash and Python.

Python syntax. Use Python anywhere.

Bash compatible. Shell commands work.

Tab completion. Context-aware.

Cross-platform. Works everywhere Python does.

# CAVEATS

Python required. Different from bash. Learning curve.

# HISTORY

**xonsh** was created to combine the power of Python with the familiarity of shell scripting.
