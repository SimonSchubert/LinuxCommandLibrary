# TLDR

**Show teddy bear**

```xteddy```

**Different image**

```xteddy -f [image.png]```

**Start at position**

```xteddy -geometry [+100+100]```

# SYNOPSIS

**xteddy** [_-f image_] [_-geometry pos_] [_options_]

# PARAMETERS

**-f** _FILE_
> Image file.

**-geometry** _POS_
> Window position.

**-display** _DPY_
> X display.

# DESCRIPTION

**xteddy** shows a teddy bear. It's a desktop mascot.

Desktop companion. Friendly bear.

Draggable window. Move around.

Click to quit. Simple interaction.

X11 toy. Fun application.

# CAVEATS

X11 only. Just for fun. Nostalgia.

# HISTORY

**xteddy** is a classic X11 desktop toy displaying a teddy bear that can be dragged around the screen.
