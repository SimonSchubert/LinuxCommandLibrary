# TAGLINE

Convert XV thumbnails to PPM format

# TLDR

**Convert XV thumbnail to PPM**

```xvminitoppm [input.xv] > [output.ppm]```

# SYNOPSIS

**xvminitoppm** [_file_]

# DESCRIPTION

**xvminitoppm** converts XV visual schnauzer thumbnail format to PPM. XV was a popular Unix image viewer in the 1990s. Part of Netpbm toolkit.
