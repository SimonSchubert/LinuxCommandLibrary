# TLDR

**Convert XWD to PNM**

```xwdtopnm [input.xwd] > [output.pnm]```

# SYNOPSIS

**xwdtopnm** [_file_]

# DESCRIPTION

**xwdtopnm** converts X Window Dump (XWD) images to PNM format. XWD is created by xwd(1) for X11 window screenshots. Part of Netpbm toolkit.
