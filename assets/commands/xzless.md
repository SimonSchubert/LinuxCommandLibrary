# TAGLINE

View xz compressed files with paging

# TLDR

**View xz compressed file**

```xzless [file.xz]```

**View multiple files**

```xzless [file1.xz] [file2.xz]```

# SYNOPSIS

**xzless** [_options_] [_files_...]

# DESCRIPTION

**xzless** views xz compressed files with paging. Decompresses and pipes to less. Supports all less navigation commands. Use for browsing large compressed log files.

# INSTALL

```apt: sudo apt install xz-utils```

```dnf: sudo dnf install xz```

```pacman: sudo pacman -S xz```

```apk: sudo apk add xz```

```zypper: sudo zypper install xz```

```brew: brew install xz```

```nix: nix profile install nixpkgs#xz```
