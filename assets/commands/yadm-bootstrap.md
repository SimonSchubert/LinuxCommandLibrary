# TAGLINE

Run yadm post-clone bootstrap script

# TLDR

**Run bootstrap script**

```yadm bootstrap```

# SYNOPSIS

**yadm** **bootstrap**

# DESCRIPTION

**yadm bootstrap** executes the bootstrap script (~/.config/yadm/bootstrap). Runs post-clone setup like installing packages, setting permissions, or configuring applications. Script must be executable.
