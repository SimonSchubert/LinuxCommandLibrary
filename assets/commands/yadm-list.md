# TAGLINE

List yadm tracked dotfiles

# TLDR

**List tracked files**

```yadm list```

**List with attributes**

```yadm list -a```

# SYNOPSIS

**yadm** **list** [_options_]

# PARAMETERS

**-a**
> Include attributes.

# DESCRIPTION

**yadm list** shows files tracked by yadm. Displays all dotfiles under version control. Useful for auditing managed files.

# INSTALL

```pacman: sudo pacman -S yadm```

```apk: sudo apk add yadm```

```zypper: sudo zypper install yadm```

```brew: brew install yadm```

```nix: nix profile install nixpkgs#yadm```
