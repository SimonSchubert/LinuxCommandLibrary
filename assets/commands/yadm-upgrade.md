# TLDR

**Upgrade yadm configuration**

```yadm upgrade```

# SYNOPSIS

**yadm** **upgrade** [_options_]

# PARAMETERS

**-f**
> Force upgrade.

# DESCRIPTION

**yadm upgrade** migrates yadm configuration to current version. Updates paths and formats when upgrading yadm. Run after yadm version upgrades to ensure compatibility.
