# TAGLINE

KDE drop-down terminal emulator

# TLDR

**Start** yakuake

```yakuake```

# SYNOPSIS

**yakuake** [_OPTIONS_]

# PARAMETERS

**F12**
> Toggle visibility

**F11**
> Toggle fullscreen

**Ctrl+Shift+t**
> New tab

**Shift+Arrow**
> Switch tabs

**Ctrl+)/Ctrl+(**
> Create terminal splits

**Ctrl+Tab**
> Switch between splits

# DESCRIPTION

**yakuake** is a drop-down terminal emulator for KDE. It slides down from the top of the screen when activated, similar to Quake-style consoles.

The terminal supports tabs, split views, and extensive customization through KDE settings.

# CAVEATS

Designed for KDE but may work on other desktops. Requires KDE frameworks libraries.
