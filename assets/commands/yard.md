# TLDR

**Generate docs**

```yard doc```

**Start server**

```yard server```

**List undocumented**

```yard stats --list-undoc```

**Generate for gems**

```yard gems```

**Show documentation**

```yard ri [ClassName]```

# SYNOPSIS

**yard** _command_ [_options_]

# PARAMETERS

**doc**
> Generate docs.

**server**
> Start server.

**stats**
> Show statistics.

**gems**
> Document gems.

**ri**
> Show documentation.

**--list-undoc**
> Undocumented items.

# DESCRIPTION

**yard** documents Ruby code. It generates HTML docs.

API documentation. From comments.

Markdown support. Rich formatting.

Templates. Customizable output.

Server mode. Live preview.

# CAVEATS

Ruby-specific. YARD syntax in comments. Gem required.

# HISTORY

**YARD** (Yay! A Ruby Documentation Tool) was created as a modern replacement for RDoc.
