# TLDR

**Show why package is installed**

```yarn why [package]```

# SYNOPSIS

**yarn** **why** _package_

# DESCRIPTION

**yarn why** explains why a package is installed. Shows which packages depend on it. Useful for understanding dependency chains and auditing node_modules.
