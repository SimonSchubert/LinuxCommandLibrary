# TLDR

**Convert Bennet Yee face to PBM**

```ybmtopbm [input.ybm] > [output.pbm]```

# SYNOPSIS

**ybmtopbm** [_file_]

# DESCRIPTION

**ybmtopbm** converts Bennet Yee's "face" format to PBM. YBM was used for simple face images. Part of Netpbm toolkit.
