# TAGLINE

Combine YUV component files to PPM

# TLDR

**Combine YUV component files to PPM**

```yuvsplittoppm [base] [width] [height] > [output.ppm]```

# SYNOPSIS

**yuvsplittoppm** _basename_ _width_ _height_

# PARAMETERS

_basename_
> Base name of .Y, .U, .V files.

_width_
> Image width.

_height_
> Image height.

# DESCRIPTION

**yuvsplittoppm** combines separate Y, U, V component files into a single PPM image. Inverse of ppmtoyuvsplit. Part of Netpbm toolkit.
