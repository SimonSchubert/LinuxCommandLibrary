# TLDR

**Show integration analytics**

```zapier analytics```

# SYNOPSIS

**zapier** **analytics**

# DESCRIPTION

**zapier analytics** displays usage analytics for your Zapier integration. Shows usage statistics and performance metrics. Part of Zapier CLI for integration development.
