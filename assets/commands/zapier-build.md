# TAGLINE

Build Zapier integration for deployment

# TLDR

**Build integration**

```zapier build```

# SYNOPSIS

**zapier** **build**

# DESCRIPTION

**zapier build** compiles a Zapier integration for deployment. Builds source files into deployable format. Run before push to verify build succeeds.
