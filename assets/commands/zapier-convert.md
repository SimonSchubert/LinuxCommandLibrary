# TLDR

**Convert legacy integration**

```zapier convert [app_id] [path]```

**Convert with version**

```zapier convert [app_id] [path] --version [version]```

# SYNOPSIS

**zapier** **convert** _app_id_ _path_ [_options_]

# PARAMETERS

**--version** _version_
> Specific version to convert.

# DESCRIPTION

**zapier convert** migrates legacy Web Builder integrations to CLI format. Converts visual integrations to code-based projects for advanced development.
