# TAGLINE

View Zapier integration deployment history

# TLDR

**Show version history**

```zapier history```

# SYNOPSIS

**zapier** **history**

# DESCRIPTION

**zapier history** displays deployment history for your Zapier integration. Shows version numbers, dates, and deployment notes.
