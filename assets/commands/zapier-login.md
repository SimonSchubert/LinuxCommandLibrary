# TAGLINE

Authenticate with Zapier platform

# TLDR

**Login to Zapier**

```zapier login```

**Login with SSO**

```zapier login --sso```

# SYNOPSIS

**zapier** **login** [_options_]

# PARAMETERS

**--sso**
> Use SSO authentication.

# DESCRIPTION

**zapier login** authenticates with the Zapier platform. Opens browser for authentication and stores credentials locally. Required before using other Zapier CLI commands.
