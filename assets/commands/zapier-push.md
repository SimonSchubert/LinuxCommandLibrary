# TAGLINE

Deploy integration to Zapier

# TLDR

**Push integration**

```zapier push```

# SYNOPSIS

**zapier** **push**

# DESCRIPTION

**zapier push** deploys your integration to Zapier. Uploads built integration and creates a new version. Run after zapier build or let push build automatically.
