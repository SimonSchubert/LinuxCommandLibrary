# TAGLINE

Validate Zapier integration configuration

# TLDR

**Validate integration**

```zapier validate```

# SYNOPSIS

**zapier** **validate**

# DESCRIPTION

**zapier validate** checks integration for errors. Validates schema, required fields, and configuration. Run before push to catch issues early.
