# TLDR

**Open file**

```zeditor [file]```

**Open at line**

```zeditor [file]:[line]```

**Open multiple files**

```zeditor [file1] [file2]```

**New window**

```zeditor --new [file]```

# SYNOPSIS

**zeditor** [_--new_] [_options_] [_files_]

# PARAMETERS

**--new**
> New window.

**--wait**
> Wait for close.

**--help**
> Show help.

# DESCRIPTION

**zeditor** opens Zed editor. It's the CLI launcher.

Modern editor. Fast and minimal.

Rust-based. High performance.

Collaborative. Real-time editing.

AI features. Code assistance.

# CAVEATS

Zed installation required. Newer editor. Platform support varies.

# HISTORY

**zeditor** is the command-line launcher for **Zed**, a modern code editor built in Rust.
