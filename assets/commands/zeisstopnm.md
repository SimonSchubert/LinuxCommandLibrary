# TLDR

**Convert Zeiss confocal to PNM**

```zeisstopnm [input.zeiss] > [output.pnm]```

# SYNOPSIS

**zeisstopnm** [_file_]

# DESCRIPTION

**zeisstopnm** converts Zeiss confocal microscope image format to PNM. Used in scientific imaging applications. Part of Netpbm toolkit.
