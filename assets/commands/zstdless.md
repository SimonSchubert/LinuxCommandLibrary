# TLDR

**View zstd compressed file**

```zstdless [file.zst]```

# SYNOPSIS

**zstdless** [_files_...]

# DESCRIPTION

**zstdless** views zstd compressed files with paging. Decompresses and pipes to less. Supports full less navigation. Use for browsing compressed log files and data.
